    Bounded Type Parameter¶

    нғҖмһ… нҢҢлқјлҜён„°м—җ мғҒн•ңмқ„ м§Җм •н•ҳл©ҙ н•ҙлӢ№ нғҖмһ…мқҳ л©”м„ңл“ңлҘј мӮ¬мҡ©н•  мҲҳ мһҲлӢӨ.

    // TлҠ” Comparableмқ„ кө¬нҳ„н•ң нғҖмһ…л§Ң л°ӣлҠ”лӢӨ
public static <T extends Comparable<T>> T max(T a, T b) {
    return a.compareTo(b) >= 0 ? a : b;
}

    м—¬лҹ¬ л°”мҡҙл“ңлҘј лҸҷмӢңм—җ кұё мҲҳ мһҲлӢӨ. нҒҙлһҳмҠӨк°Җ лЁјм Җ, мқён„°нҺҳмқҙмҠӨк°Җ л’Өм—җ мҳЁлӢӨ.

    // TлҠ” Numberмқҳ н•ҳмң„ нҒҙлһҳмҠӨмқҙл©ҙм„ң Comparableмқ„ кө¬нҳ„н•ҙм•ј н•ңлӢӨ
public static <T extends Number & Comparable<T>> T max(T a, T b) {
    return a.compareTo(b) >= 0 ? a : b;
}

    мһ¬к·Җм Ғ нғҖмһ… л°”мҡҙл“ң (Recursive Type Bound)¶

    <T extends Comparable<T>> нҳ•нғңлҘј мһ¬к·Җм Ғ нғҖмһ… л°”мҡҙл“ңлқј н•ңлӢӨ. мһҗкё° мһҗмӢ кіј л№„көҗ к°ҖлҠҘн•ң нғҖмһ…л§Ң л°ӣкІ лӢӨлҠ” мқҳлҜёлӢӨ.

    мқҙ нҢЁн„ҙмқҖ Comparable, Builder, Enum л“ұм—җм„ң мһҗмЈј лӮҳмҳЁлӢӨ.

    // Enumмқҳ мӢӨм ң м„ м–ё
public abstract class Enum<E extends Enum<E>> implements Comparable<E> { }

    Builder нҢЁн„ҙм—җм„ң мғҒмҶҚмқ„ м§Җмӣҗн•  л•Ң м“°лҠ” л°©лІ•мқҙлӢӨ.

    public abstract class Pizza {
    final Set<String> toppings;

    abstract static class Builder<T extends Builder<T>> {
        Set<String> toppings = new HashSet<>();

        public T addTopping(String topping) {
            toppings.add(topping);
            return self();
        }

        // н•ҳмң„ нҒҙлһҳмҠӨм—җм„ң thisлҘј л°ҳнҷҳ
        protected abstract T self();
        public abstract Pizza build();
    }

    Pizza(Builder<?> builder) {
        toppings = new HashSet<>(builder.toppings);
    }
}

public class Calzone extends Pizza {
    private final boolean sauceInside;

    public static class Builder extends Pizza.Builder<Builder> {
        private boolean sauceInside = false;

        public Builder sauceInside() {
            sauceInside = true;
            return this;
        }

        @Override
        protected Builder self() { return this; }

        @Override
        public Calzone build() { return new Calzone(this); }
    }

    private Calzone(Builder builder) {
        super(builder);
        sauceInside = builder.sauceInside;
    }
}

Calzone calzone = new Calzone.Builder()
    .addTopping("Ham")      // Calzone.Builder л°ҳнҷҳ (Pizza.Builderк°Җ м•„лӢҳ)
    .sauceInside()

    PECS (Producer Extends, Consumer Super)¶

    мҷҖмқјл“ңм№ҙл“ңлҘј м“ё л•Ң extendsмҷҖ super мӨ‘ лӯҳ мҚЁм•ј н•ҳлҠ”м§Җ нҢҗлӢЁн•ҳлҠ” к·ңм№ҷмқҙлӢӨ.

    • лҚ°мқҙн„°лҘј мқҪкё°л§Ң н•ҳл©ҙ(produce) ? extends T
    • лҚ°мқҙн„°лҘј м“°кё°л§Ң н•ҳл©ҙ(consume) ? super T
    • мқҪкі  м“°кё° л‘ҳ лӢӨ н•ҳл©ҙ мҷҖмқјл“ңм№ҙл“ңлҘј м“°м§Җ м•ҠлҠ”лӢӨ
    // srcм—җм„ң мқҪкі (produce), dstм—җ м“ҙлӢӨ(consume)
public static <T> void copy(List<? extends T> src, List<? super T> dst) {
    for (T item : src) {
        dst.add(item);
    }
}

    Collections.copyмқҳ мӢӨм ң мӢңк·ёлӢҲмІҳк°Җ мқҙ нҳ•нғңлӢӨ.

    мҷң мқҙл ҮкІҢ н•ҙм•ј н•ҳлҠ”к°Җ¶

    List<? extends Number>м—җлҠ” List<Integer>, List<Double> л“ұмқҙ мҳ¬ мҲҳ мһҲлӢӨ. м—¬кё°м—җ к°’мқ„ л„Јмңјл©ҙ мӢӨм ң лҰ¬мҠӨнҠё нғҖмһ…кіј м¶©лҸҢн•  мҲҳ мһҲмңјлҜҖлЎң мқҪкё°л§Ң к°ҖлҠҘн•ҳлӢӨ.

    List<? extends Number> numbers = new ArrayList<Integer>();
Number n = numbers.get(0);   // OK вҖ” NumberлЎң мқҪмқ„ мҲҳ мһҲлӢӨ
// numbers.add(1);            // м»ҙнҢҢмқј м—җлҹ¬ вҖ” Integerмқём§Җ Doubleмқём§Җ м•Ң мҲҳ м—ҶлӢӨ

    List<? super Integer>м—җлҠ” List<Integer>, List<Number>, List<Object> л“ұмқҙ мҳ¬ мҲҳ мһҲлӢӨ. IntegerлҠ” мқҙ лӘЁл“  нғҖмһ…мқҳ н•ҳмң„мқҙлҜҖлЎң м“°кё°к°Җ к°ҖлҠҘн•ҳлӢӨ. мқҪмқ„ л•ҢлҠ” ObjectлЎңл§Ң л°ӣмқ„ мҲҳ мһҲлӢӨ.

    List<? super Integer> list = new ArrayList<Number>();
list.add(1);                  // OK вҖ” IntegerлҠ” л„Јмқ„ мҲҳ мһҲлӢӨ
Object obj = list.get(0);    // OK вҖ” ObjectлЎңл§Ң мқҪмқ„ мҲҳ мһҲлӢӨ
// Integer i = list.get(0);   // м»ҙнҢҢмқј м—җлҹ¬

    мӢӨл¬ҙ м Ғмҡ©¶

    ComparatorлҘј нҢҢлқјлҜён„°лЎң л°ӣмқ„ л•Ң нқ”нһҲ мӢӨмҲҳн•ҳлҠ” л¶Җл¶„мқҙлӢӨ.

    // мқҙл ҮкІҢ н•ҳл©ҙ Comparator<String>л§Ң л°ӣмқ„ мҲҳ мһҲлӢӨ
public static <T> void sort(List<T> list, Comparator<T> comparator) { }

// мқҙл ҮкІҢ н•ҳл©ҙ Comparator<Object>лҸ„ л°ӣмқ„ мҲҳ мһҲлӢӨ
// ComparatorлҠ” TлҘј consumeн•ҳлҜҖлЎң superлҘј м“ҙлӢӨ
public static <T> void sort(List<T> list, Comparator<? super T> comparator) { }

    Collections.sortмқҳ мӢӨм ң мӢңк·ёлӢҲмІҳк°Җ Comparator<? super T>лҘј л°ӣлҠ”лӢӨ. Comparator<Object>лЎң лӘЁл“  нғҖмһ…мқ„ л№„көҗн•  мҲҳ мһҲм–ҙм•ј н•ҳлҠ” мғҒнҷ©мқҙ мһҲкё° л•Ңл¬ёмқҙлӢӨ.

    // Comparator<Object>лҘј String лҰ¬мҠӨнҠём—җлҸ„ м“ё мҲҳ мһҲлӢӨ
Comparator<Object> byHashCode = Comparator.comparingInt(Object::hashCode);
List<String> names = Arrays.asList("Kim", "Lee", "Park");
names.sort(byHashCode); // Comparator<? super String>мқҙлҜҖлЎң к°ҖлҠҘ

    кіөліҖ, л°ҳкіөліҖ, л¬ҙкіөліҖ¶

    л°°м—ҙкіј м ңл„ӨлҰӯмқҳ нғҖмһ… кҙҖкі„к°Җ лӢӨлҘҙлӢӨ. мқҙкұё лӘЁлҘҙл©ҙ лҹ°нғҖмһ„ м—җлҹ¬лҘј л§ҢлӮҳкІҢ лҗңлӢӨ.

    л°°м—ҙмқҖ кіөліҖ(covariant)

    Integerк°Җ Numberмқҳ н•ҳмң„ нғҖмһ…мқҙл©ҙ, Integer[]лҸ„ Number[]мқҳ н•ҳмң„ нғҖмһ…мқҙлӢӨ.

    Number[] numbers = new Integer[10]; // м»ҙнҢҢмқј OK
numbers[0] = 3.14;                  // лҹ°нғҖмһ„м—җ ArrayStoreException

    м»ҙнҢҢмқјмқҖ лҗҳм§Җл§Ң лҹ°нғҖмһ„м—җ н„°м§„лӢӨ. л°°м—ҙмқҳ кіөліҖм„ұмқҖ м ңл„ӨлҰӯ мқҙм „мқҳ м„Өкі„ кІ°м •мқҙлӢӨ.

    м ңл„ӨлҰӯмқҖ л¬ҙкіөліҖ(invariant)

    List<Integer>лҠ” List<Number>мқҳ н•ҳмң„ нғҖмһ…мқҙ м•„лӢҲлӢӨ.

    // м»ҙнҢҢмқј м—җлҹ¬
// List<Number> numbers = new ArrayList<Integer>();

// мҷҖмқјл“ңм№ҙл“ңлҘј мҚЁм•ј н•ңлӢӨ
List<? extends Number> numbers = new ArrayList<Integer>(); // OK

    л¬ҙкіөліҖмқҙкё° л•Ңл¬ём—җ м»ҙнҢҢмқј мӢңм җм—җ нғҖмһ… м•Ҳм „м„ұмқ„ ліҙмһҘн•ңлӢӨ. л°°м—ҙмІҳлҹј лҹ°нғҖмһ„м—җ н„°м§ҖлҠ” мқјмқҙ м—ҶлӢӨ.

    мҷҖмқјл“ңм№ҙл“ңлЎң кіөліҖ/л°ҳкіөліҖ н‘ңнҳ„

    • ? extends T вҖ” кіөліҖ. мқҪкё° м „мҡ©мңјлЎң н•ҳмң„ нғҖмһ… н—Ҳмҡ©
    • ? super T вҖ” л°ҳкіөліҖ. м“°кё° м „мҡ©мңјлЎң мғҒмң„ нғҖмһ… н—Ҳмҡ©

    мқҙ кҙҖкі„лҘј мқҙн•ҙн•ҳл©ҙ PECSк°Җ мһҗм—°мҠӨлҹҪкІҢ л”°лқјмҳЁлӢӨ.

    нһҷ мҳӨм—ј (Heap Pollution)¶

    л§Өк°ңліҖмҲҳнҷ”лҗң нғҖмһ…мқҳ ліҖмҲҳк°Җ мӢӨм ңлЎңлҠ” лӢӨлҘё нғҖмһ…мқҳ к°қмІҙлҘј м°ёмЎ°н•ҳлҠ” мғҒнғңлӢӨ. мЈјлЎң к°ҖліҖмқёмһҗ(varargs)мҷҖ м ңл„ӨлҰӯмқ„ мЎ°н•©н•  л•Ң л°ңмғқн•ңлӢӨ.

    public static void faultyMethod(List<String>... stringLists) {
    Object[] array = stringLists;          // List<String>[]мқҙ Object[]лЎң ліҖнҷҳ
    List<Integer> intList = Arrays.asList(42);
    array[0] = intList;                     // нһҷ мҳӨм—ј л°ңмғқ

    @SafeVarargs¶

    л©”м„ңл“ңк°Җ к°ҖліҖмқёмһҗ л°°м—ҙмқ„ нҶөн•ҙ нһҷ мҳӨм—јмқ„ мқјмңјнӮӨм§Җ м•ҠлҠ”лӢӨкі  ліҙмһҘн•  л•Ң м“ҙлӢӨ. м»ҙнҢҢмқјлҹ¬ кІҪкі лҘј м ңкұ°н•ңлӢӨ.

@SafeVarargs

    Bridge Method¶

    м ңл„ӨлҰӯмқ„ мғҒмҶҚн•  л•Ң м»ҙнҢҢмқјлҹ¬к°Җ мһҗлҸҷмңјлЎң мғқм„ұн•ҳлҠ” л©”м„ңл“ңлӢӨ. Type ErasureлЎң мқён•ҙ л¶ҖлӘЁмҷҖ мһҗмӢқмқҳ л©”м„ңл“ң мӢңк·ёлӢҲмІҳк°Җ лӢ¬лқјм§ҖлҠ” кІғмқ„ ліҙм •н•ңлӢӨ.

    private T data;
    public void setData(T data) {

    }

    public void setData(String data) {

    Super Type Token¶

    Neal Gafterк°Җ м ңм•Ҳн•ң нҢЁн„ҙмңјлЎң, мқөлӘ… нҒҙлһҳмҠӨмқҳ м ңл„ӨлҰӯ нғҖмһ… м •ліҙк°Җ л°”мқҙнҠёмҪ”л“ңм—җ лӮЁлҠ” м җмқ„ мқҙмҡ©н•ңлӢӨ.

    // к°„мҶҢнҷ”н•ң кө¬нҳ„
public abstract class TypeReference<T> {
    private final Type type;

    protected TypeReference() {
        Type superClass = getClass().getGenericSuperclass();
        this.type = ((ParameterizedType) superClass).getActualTypeArguments()[0];
    }

    public Type getType() { return type; }
}

// мӮ¬мҡ© вҖ” л°ҳл“ңмӢң мқөлӘ… нҒҙлһҳмҠӨ(мӨ‘кҙ„нҳё {})лЎң мғқм„ұн•ҙм•ј н•ңлӢӨ
TypeReference<List<String>> ref = new TypeReference<List<String>>() {};
System.out.println(ref.getType());
// java.util.List<java.lang.String>

    Jacksonмқҳ TypeReference, Guiceмқҳ TypeLiteralмқҙ мқҙ нҢЁн„ҙмқҳ кө¬нҳ„мІҙлӢӨ.
