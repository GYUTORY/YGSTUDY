OAuthк°Җ м—Ҷмңјл©ҙ мӮ¬мҡ©мһҗмқҳ Google л№„л°ҖлІҲнҳёлҘј м§Ғм ‘ л°ӣм•„м„ң лЎңк·ёмқён•ҙм•ј н•ңлӢӨ. лӢ№м—°нһҲ л§җмқҙ м•Ҳ лҗңлӢӨ. OAuthлҠ” мқҙ л¬ём ңлҘј н•ҙкІ°н•ңлӢӨ. мӮ¬мҡ©мһҗк°Җ Google лЎңк·ёмқё нҷ”л©ҙм—җм„ң м§Ғм ‘ мқёмҰқн•ҳкі , мҡ°лҰ¬ м•ұм—җлҠ” "мәҳлҰ°лҚ” мқҪкё° к¶Ңн•ң"мқҙ лӢҙкёҙ нҶ нҒ°л§Ң м „лӢ¬лҗңлӢӨ.

    Authorization Code Grant + PKCE¶

    SPAлӮҳ лӘЁл°”мқј м•ұмІҳлҹј client_secretмқ„ м•Ҳм „н•ҳкІҢ ліҙкҙҖн•  мҲҳ м—ҶлҠ” нҷҳкІҪм—җм„ң м“ҙлӢӨ. OAuth 2.1м—җм„ңлҠ” лӘЁл“  нҒҙлқјмқҙм–ёнҠём—җ PKCEлҘј н•„мҲҳлЎң мҡ”кө¬н•ңлӢӨ.

    лҸҷмһ‘ мӣҗлҰ¬:

    1. нҒҙлқјмқҙм–ёнҠёк°Җ code_verifier(лһңлҚӨ л¬ёмһҗм—ҙ)лҘј мғқм„ұ
    2. code_verifierлҘј SHA-256 н•ҙмӢұн•ҙм„ң code_challengeлҘј л§Ңл“Ұ
    3. мқёмҰқ мҡ”мІӯ мӢң code_challengeлҘј н•Ёк»ҳ м „мҶЎ
    4. нҶ нҒ° көҗнҷҳ мӢң мӣҗліё code_verifierлҘј м „мҶЎ
    5. мқёмҰқ м„ңлІ„к°Җ code_verifierлҘј н•ҙмӢұн•ҙм„ң мІҳмқҢ л°ӣмқҖ code_challengeмҷҖ л№„көҗ

    Implicit Grant (deprecated)¶

    мқҙ л°©мӢқмқҖ лҚ” мқҙмғҒ м“°л©ҙ м•Ҳ лҗңлӢӨ. OAuth 2.1м—җм„ң кіөмӢқм ҒмңјлЎң м ңкұ°лҗҳм—ҲлӢӨ.

    мқёмҰқ мҪ”л“ң көҗнҷҳ лӢЁкі„ м—Ҷмқҙ access_tokenмқ„ URL fragment(#access_token=xxx)лЎң л°”лЎң м „лӢ¬н•ҳлҠ” л°©мӢқмқҙм—ҲлӢӨ. SPAлҘј мң„н•ҙ л§Ңл“Өм–ҙмЎҢм§Җл§Ң, нҶ нҒ°мқҙ лёҢлқјмҡ°м Җ нһҲмҠӨнҶ лҰ¬м—җ лӮЁкі , Referer н—ӨлҚ”лЎң мң м¶ңлҗ  мҲҳ мһҲлҠ” л¬ём ңк°Җ мһҲлӢӨ.

    SPAм—җм„ңлҠ” Authorization Code + PKCEлҘј м“°л©ҙ лҗңлӢӨ. л Ҳкұ°мӢң мӢңмҠӨн…ңм—җм„ң Implicit GrantлҘј л°ңкІ¬н•ҳл©ҙ л§Ҳмқҙк·ёл Ҳмқҙм…ҳн•ҙм•ј н•ңлӢӨ.

    redirect_uri кІҖмҰқ вҖ” к°ҖмһҘ нқ”н•ң мӢӨмҲҳ¶

    Authorization Code нғҲм·Ёмқҳ к°ҖмһҘ нқ”н•ң мӣҗмқёмқҙ redirect_uri кІҖмҰқ мӢӨнҢЁлӢӨ.

    л¬ём ң мғҒнҷ©¶

    мқёмҰқ м„ңлІ„м—җм„ң redirect_uriлҘј лҠҗмҠЁн•ҳкІҢ кІҖмҰқн•ҳл©ҙ кіөкІ©мһҗк°Җ мқҙл ҮкІҢ мҡ”мІӯн•  мҲҳ мһҲлӢӨ:

    GET /authorize?
  response_type=code
  &client_id=legitimate-app
  &redirect_uri=https://legitimate-app.com.attacker.com/callback
  &scope=openid profile

    мқёмҰқ м„ңлІ„к°Җ prefixл§Ң мІҙнҒ¬н•ҳл©ҙ legitimate-app.com.attacker.comлҸ„ нҶөкіјн•ңлӢӨ. мӮ¬мҡ©мһҗлҠ” м •мғҒм Ғмқё лЎңк·ёмқё нҷ”л©ҙмқ„ ліҙкі  мқёмҰқн•ҳм§Җл§Ң, authorization codeлҠ” кіөкІ©мһҗ м„ңлІ„лЎң к°„лӢӨ.

    м§Җмјңм•ј н•  кІғ¶

    • redirect_uriлҠ” м •нҷ•нһҲ мқјм№ҳ(exact match) н•ҙм•ј н•ңлӢӨ. мҷҖмқјл“ңм№ҙл“ңлӮҳ prefix л§Өм№ӯмқҖ м“°л©ҙ м•Ҳ лҗңлӢӨ
    • мқёмҰқ м„ңлІ„м—җ л“ұлЎқлҗң redirect_uriмҷҖ мҡ”мІӯмқҳ redirect_uriк°Җ л¬ёмһҗм—ҙ лӢЁмң„лЎң лҸҷмқјн•ҙм•ј н•ңлӢӨ
    • к°ңл°ң нҷҳкІҪмҡ© http://localhost URIлҠ” н”„лЎңлҚ•м…ҳ м„Өм •м—җ м ҲлҢҖ лӮЁкё°л©ҙ м•Ҳ лҗңлӢӨ
    • мқёмҰқ м„ңлІ„лҘј м§Ғм ‘ кө¬м¶•н•ҳлҠ” кІҪмҡ°, redirect_uri л№„көҗ лЎңм§Ғм—җ URL м •к·ңнҷ”(normalize)лҘј л„Јм§Җ л§Ҳлқј. path traversal(/callback/../admin/callback)лЎң мҡ°нҡҢлӢ№н•  мҲҳ мһҲлӢӨ

    PKCE кө¬нҳ„м—җм„ң кІӘлҠ” л¬ём ңл“Ө¶

    code_verifier м„ём…ҳ мң мӢӨ¶

    PKCEм—җм„ң к°ҖмһҘ л§Һмқҙ кІӘлҠ” л¬ём ңлӢӨ. мқёмҰқ мҡ”мІӯ мӢңм җм—җ code_verifierлҘј м„ём…ҳм—җ м ҖмһҘн•ҳкі , мҪңл°ұм—җм„ң кәјлӮҙ м“°лҠ”лҚ°, мқҙ мӮ¬мқҙм—җ м„ём…ҳмқҙ лӮ м•„к°ҖлҠ” кІҪмҡ°к°Җ мһҲлӢӨ.

    л°ңмғқн•ҳлҠ” мғҒнҷ©:

    • лЎңл“ңл°ёлҹ°м„ң л’Өм—җ м„ңлІ„к°Җ м—¬лҹ¬ лҢҖ мһҲкі  м„ём…ҳмқҙ sticky лҗҳм§Җ м•Ҡмқ„ л•Ң
    • мқёмҰқ нҷ”л©ҙм—җм„ң мӢңк°„мқҙ мҳӨлһҳ кұёл Өм„ң м„ём…ҳмқҙ л§ҢлЈҢлҗ  л•Ң
    • лӘЁл°”мқј м•ұм—җм„ң мҷёл¶Җ лёҢлқјмҡ°м ҖлЎң мқёмҰқн•ҳкі  лҸҢм•„мҳ¬ л•Ң м•ұмқҙ л©”лӘЁлҰ¬м—җм„ң лӮҙл Өк°„ кІҪмҡ°

    н•ҙкІ° л°©лІ•:

    • м„ңлІ„ м„ём…ҳ лҢҖмӢ  Redis к°ҷмқҖ мҷёл¶Җ м ҖмһҘмҶҢм—җ code_verifierлҘј м ҖмһҘн•ңлӢӨ. keyлҠ” state нҢҢлқјлҜён„°лҘј м“°л©ҙ лҗңлӢӨ
    • SPAлқјл©ҙ sessionStorageм—җ м ҖмһҘн•ңлӢӨ. localStorageлҠ” нғӯ к°„ кіөмң лҗҳлҜҖлЎң м—¬лҹ¬ нғӯм—җм„ң лҸҷмӢң лЎңк·ёмқё мӢң кј¬мқј мҲҳ мһҲлӢӨ
    • лӘЁл°”мқј м•ұмқҖ Keychain/Keystoreм—җ м ҖмһҘн•ҳкі , м•ұ ліөмӣҗ мӢң мқҪмқ„ мҲҳ мһҲкІҢ н•ҙм•ј н•ңлӢӨ

    S256мқ„ м§Җмӣҗн•ҳм§Җ м•ҠлҠ” м„ңлІ„¶

    PKCEмқҳ code_challenge_methodм—җлҠ” plainкіј S256 л‘җ к°Җм§Җк°Җ мһҲлӢӨ. plainмқҖ code_verifierлҘј к·ёлҢҖлЎң ліҙлӮҙлҠ” кұ°лқј ліҙм•ҲмғҒ мқҳлҜёк°Җ кұ°мқҳ м—ҶлӢӨ. л°ҳл“ңмӢң S256(SHA-256 н•ҙмӢң)мқ„ мҚЁм•ј н•ңлӢӨ.

    л¬ём ңлҠ” мқјл¶Җ мҳӨлһҳлҗң мқёмҰқ м„ңлІ„к°Җ S256мқ„ м§Җмӣҗн•ҳм§Җ м•ҠлҠ” кІҪмҡ°лӢӨ. мқҙл•Ң plainмңјлЎң fallbackн•ҳл©ҙ PKCEлҘј м•Ҳ м“°лҠ” кІғкіј лӢӨлҘј л°” м—ҶлӢӨ. м„ңлІ„к°Җ S256мқ„ м§Җмӣҗн•ҳм§Җ м•Ҡмңјл©ҙ н•ҙлӢ№ м„ңлІ„мқҳ OAuth кө¬нҳ„ мһҗмІҙлҘј мқҳмӢ¬н•ҙм•ј н•ңлӢӨ.

    нҶ нҒ° к°ұмӢ  мӢң race condition¶

    access_token л§ҢлЈҢ нӣ„ refresh_tokenмңјлЎң к°ұмӢ н•  л•Ң, лҸҷмӢңм—җ м—¬лҹ¬ API мҡ”мІӯмқҙ л°ңмғқн•ҳл©ҙ л¬ём ңк°Җ мғқкёҙлӢӨ.

    мӢңлӮҳлҰ¬мҳӨ¶

    1. мҡ”мІӯ Aк°Җ 401 мқ‘лӢөмқ„ л°ӣмқҢ (access_token л§ҢлЈҢ)
    2. мҡ”мІӯ BлҸ„ 401 мқ‘лӢөмқ„ л°ӣмқҢ
    3. мҡ”мІӯ Aк°Җ refresh_tokenмңјлЎң мғҲ access_token мҡ”мІӯ
    4. мҡ”мІӯ BлҸ„ к°ҷмқҖ refresh_tokenмңјлЎң мғҲ access_token мҡ”мІӯ
    5. мқёмҰқ м„ңлІ„к°Җ refresh_token rotationмқ„ м Ғмҡ©н•ҳкі  мһҲмңјл©ҙ, мҡ”мІӯ Aм—җм„ң refresh_tokenмқҙ көҗмІҙлҗЁ
    6. мҡ”мІӯ BлҠ” мқҙлҜё нҸҗкё°лҗң refresh_tokenмқ„ мӮ¬мҡ©н•ҳкІҢ лҗҳм–ҙ мӢӨнҢЁ
    7. мқјл¶Җ мқёмҰқ м„ңлІ„лҠ” мқҙ мӢңм җм—җ н•ҙлӢ№ refresh_token familyлҘј м „л¶Җ нҸҗкё°н•Ё (ліҙм•Ҳ м •мұ…)

    н•ҙкІ° л°©лІ•¶

    // нҶ нҒ° к°ұмӢ мқ„ н•ҳлӮҳмқҳ PromiseлЎң м§Ғл ¬нҷ”
let refreshPromise = null;

async function getValidAccessToken() {
    if (isTokenValid(accessToken)) return accessToken;

    // мқҙлҜё к°ұмӢ  мӨ‘мқҙл©ҙ к°ҷмқҖ PromiseлҘј л°ҳнҷҳ
    if (!refreshPromise) {
        refreshPromise = refreshAccessToken(refreshToken)
            .then(tokens => {
                accessToken = tokens.access_token;
                if (tokens.refresh_token) {
                    refreshToken = tokens.refresh_token;
                }
                return accessToken;
            })
            .finally(() => { refreshPromise = null; });
    }
    return refreshPromise;
}

    н•өмӢ¬мқҖ к°ұмӢ  мҡ”мІӯмқ„ н•ҳлӮҳлЎң лӘЁмңјлҠ” кІғмқҙлӢӨ. мІ« лІҲм§ё к°ұмӢ  мҡ”мІӯмқҙ м§„н–ү мӨ‘мқҙл©ҙ лӮҳлЁём§ҖлҠ” к°ҷмқҖ PromiseлҘј кё°лӢӨлҰ°лӢӨ.

    м„ңлІ„ мӮ¬мқҙл“ңм—җм„ңлҸ„ л§Ҳм°¬к°Җм§ҖлӢӨ. л¶„мӮ° нҷҳкІҪмқҙлқјл©ҙ Redis lockмңјлЎң refresh_token к°ұмӢ мқ„ м§Ғл ¬нҷ”н•ҙм•ј н•ңлӢӨ.

    мҶҢм…ң лЎңк·ёмқё providerлі„ м°Ёмқҙм җ¶

    OAuth 2.0мқҖ н‘ңмӨҖмқҙм§Җл§Ң providerл§ҲлӢӨ кө¬нҳ„мқҙ лӢӨлҘҙлӢӨ. "OAuth л¶ҷмқҙл©ҙ лҒқ"мқҙлқјкі  мғқк°Ғн•ҳл©ҙ providerл§ҲлӢӨ мӮҪм§Ҳн•ңлӢӨ.

    • мөңмҙҲ лҸҷмқҳ мӢңм—җл§Ң refresh_tokenмқ„ л°ңкёүн•ңлӢӨ. мқҙнӣ„м—җлҠ” prompt=consentлҘј лӘ…мӢңн•ҙм•ј лӢӨмӢң л°ӣмқ„ мҲҳ мһҲлӢӨ

    • м•ұ м„Өм •м—җм„ң "м№ҙм№ҙмҳӨ лЎңк·ёмқё вҶ’ лҸҷмқҳн•ӯлӘ©"м—җм„ң scopeлҘј лҜёлҰ¬ м„Өм •н•ҙм•ј н•ңлӢӨ. API мҡ”мІӯм—җм„ң scopeлҘј ліҙлӮҙлҸ„ м•ұ м„Өм •м—җ м—Ҷмңјл©ҙ л¬ҙмӢңлҗңлӢӨ

    • нҡҢмӣҗ нғҲнҮҙ мӢң "л„ӨмқҙлІ„ м•„мқҙл””лЎң лЎңк·ёмқё м—°лҸҷ н•ҙм ң" APIлҘј лі„лҸ„лЎң нҳём¶ңн•ҙм•ј н•ңлӢӨ

    Spring Securityм—җм„ң providerлі„ мІҳлҰ¬¶

    Spring Security OAuth2 ClientлҘј м“°л©ҙ Google, GitHub к°ҷмқҖ CommonOAuth2ProviderлҠ” мһҗлҸҷ м„Өм •лҗңлӢӨ. н•ҳм§Җл§Ң Kakao, Naver к°ҷмқҖ көӯлӮҙ м„ңл№„мҠӨлҠ” provider м„Өм •мқ„ м§Ғм ‘ н•ҙм•ј н•ңлӢӨ.

    // providerлі„ мӮ¬мҡ©мһҗ м •ліҙ нҢҢмӢұмқҙ лӢӨлҘҙлӢӨ
public class OAuth2UserInfoFactory {
    public static OAuth2UserInfo of(String provider, Map<String, Object> attributes) {
        return switch (provider) {
            case "google" -> new GoogleUserInfo(attributes);
            case "kakao"  -> new KakaoUserInfo(

            default -> throw new IllegalArgumentException("м§Җмӣҗн•ҳм§Җ м•ҠлҠ” provider: " + provider);

}

    state нҢҢлқјлҜён„°мҷҖ CSRF¶

    stateлҠ” CSRF л°©м–ҙмҡ©мқҙлӢӨ. мқёмҰқ мҡ”мІӯ мӢң лһңлҚӨ к°’мқ„ мғқм„ұн•ҙм„ң м„ём…ҳм—җ м ҖмһҘн•ҳкі , мҪңл°ұм—җм„ң лҸҢм•„мҳЁ stateмҷҖ л№„көҗн•ңлӢӨ. мқјм№ҳн•ҳм§Җ м•Ҡмңјл©ҙ мҡ”мІӯмқ„ кұ°л¶Җн•ңлӢӨ.

    state м—Ҷмқҙ кө¬нҳ„н•ҳл©ҙ кіөкІ©мһҗк°Җ мһҗмӢ мқҳ authorization codeлҘј н”јн•ҙмһҗм—җкІҢ мЈјмһ…н•  мҲҳ мһҲлӢӨ. н”јн•ҙмһҗк°Җ кіөкІ©мһҗмқҳ кі„м •мңјлЎң лЎңк·ёмқён•ҳкІҢ лҗҳлҠ” кұ°лӢӨ. мқҙкұё "лЎңк·ёмқё CSRF"лқјкі  н•ңлӢӨ.

    нҶ нҒ° м ҖмһҘкіј ліҙм•Ҳ¶

    access_token м ҖмһҘ мң„м№ҳ¶

    нҷҳкІҪ м ҖмһҘ мң„м№ҳ мЈјмқҳмӮ¬н•ӯ
    м„ңлІ„ мӮ¬мқҙл“ң л ҢлҚ”л§Ғ м„ңлІ„ м„ём…ҳ лҳҗлҠ” Redis к°ҖмһҘ м•Ҳм „. нҶ нҒ°мқҙ лёҢлқјмҡ°м Җм—җ л…ём¶ңлҗҳм§Җ м•ҠмқҢ
    SPA л©”лӘЁлҰ¬ (ліҖмҲҳ) мғҲлЎңкі м№Ён•ҳл©ҙ лӮ м•„к°җ. silent refreshлӮҳ iframeмңјлЎң мһ¬л°ңкёү

    лӘЁл°”мқј iOS Keychain / Android Keystore лЈЁнҢ…/нғҲмҳҘ кё°кё° лҢҖмқ‘ н•„мҡ”

    localStorageм—җ access_tokenмқ„ м ҖмһҘн•ҳлҠ” мҪ”л“ңк°Җ л§Һмқҙ лҸҢм•„лӢӨлӢҲлҠ”лҚ°, XSS кіөкІ©м—җ м·Ём•Ҫн•ҳлӢӨ. XSSк°Җ н•ҳлӮҳлқјлҸ„ н„°м§Җл©ҙ нҶ нҒ°мқҙ л°”лЎң нғҲм·ЁлҗңлӢӨ.

    refresh_token кҙҖлҰ¬¶

    • refresh_tokenмқҖ access_tokenліҙлӢӨ мҲҳлӘ…мқҙ кёёлӢӨ. ліҙнҶө 7мқј~30мқј
    • refresh_token rotationмқ„ м Ғмҡ©н•ҳл©ҙ к°ұмӢ н•  л•Ңл§ҲлӢӨ мғҲ refresh_tokenмқ„ л°ңкёүн•ҳкі  мқҙм „ кІғмқ„ нҸҗкё°н•ңлӢӨ
    • нҸҗкё°лҗң refresh_tokenмқҙ мӮ¬мҡ©лҗҳл©ҙ н•ҙлӢ№ token family м „мІҙлҘј нҸҗкё°н•ҳлҠ” кІғмқҙ к¶ҢмһҘмӮ¬н•ӯмқҙлӢӨ. нҶ нҒ° нғҲм·Ё к°җм§Җмҡ©мқҙлӢӨ

    OAuth м„ңлІ„ м§Ғм ‘ кө¬м¶• vs SaaS¶

    м§Ғм ‘ кө¬м¶•н•ҳлҠ” кІҪмҡ°¶

    • мқёмҰқ/мқёк°Җ лЎңм§Ғм—җ лҢҖн•ң мҷ„м „н•ң м ңм–ҙк°Җ н•„мҡ”н•  л•Ң
    • к·ңм ң мҡ”кұҙмғҒ мӮ¬мҡ©мһҗ лҚ°мқҙн„°к°Җ мҷёл¶ҖлЎң лӮҳк°Җл©ҙ м•Ҳ лҗ  л•Ң (кёҲмңө, мқҳлЈҢ)
    • м»ӨмҠӨн…Җ grant typeмқҙлӮҳ нҠ№мҲҳн•ң нҶ нҒ° м •мұ…мқҙ н•„мҡ”н•  л•Ң

    Spring Authorization Server, Keycloak, ORY Hydra к°ҷмқҖ н”„л Ҳмһ„мӣҢнҒ¬к°Җ мһҲлӢӨ. м§Ғм ‘ RFCлҘј мқҪкі  мІҳмқҢл¶Җн„° л§Ңл“ңлҠ” кұҙ к¶ҢмһҘн•ҳм§Җ м•ҠлҠ”лӢӨ. л№ лңЁлҰ¬кё° мү¬мҡҙ ліҙм•Ҳ мҡ”кө¬мӮ¬н•ӯмқҙ л„Ҳл¬ҙ л§ҺлӢӨ.

    SaaSлҘј м“°лҠ” кІҪмҡ° (Auth0, AWS Cognito, Firebase Auth)¶

    • л№ лҘҙкІҢ м¶ңмӢңн•ҙм•ј н•ҳлҠ” м„ңл№„мҠӨ
    • мқёмҰқ мқён”„лқјлҘј мҡҙмҳҒн•  нҢҖмқҙ м—Ҷмқ„ л•Ң
    • MFA, passwordless к°ҷмқҖ кё°лҠҘмқ„ м§Ғм ‘ кө¬нҳ„н•ҳкё° л¶ҖлӢҙмҠӨлҹ¬мҡё л•Ң

    • vendor lock-inмқҙ мғқкёҙлӢӨ. лӮҳмӨ‘м—җ л§Ҳмқҙк·ёл Ҳмқҙм…ҳн•ҳл Өл©ҙ мӮ¬мҡ©мһҗ л№„л°ҖлІҲнҳё н•ҙмӢңлҘј exportн•  мҲҳ м—ҶлҠ” кІҪмҡ°к°Җ л§ҺлӢӨ

    OAuth 2.0 vs OAuth 2.1¶

    OAuth 2.1мқҖ мғҲлЎңмҡҙ н”„лЎңнҶ мҪңмқҙ м•„лӢҲлқј, кё°мЎҙ OAuth 2.0 + ліҙм•Ҳ BCP(Best Current Practice)лҘј н•ҳлӮҳлЎң н•©м№ң кІғмқҙлӢӨ.

    OpenID Connect (OIDC)¶

    OAuth 2.0мқҖ "к¶Ңн•ң мң„мһ„" н”„лЎңнҶ мҪңмқҙм§Җ "мқёмҰқ" н”„лЎңнҶ мҪңмқҙ м•„лӢҲлӢӨ. "мқҙ мӮ¬мҡ©мһҗк°Җ лҲ„кө¬мқём§Җ"лҘј нҷ•мқён•ҳлҠ” кұҙ OAuthмқҳ лІ”мң„ л°–мқҙлӢӨ.

    OpenID ConnectлҠ” OAuth 2.0 мң„м—җ мқёмҰқ л Ҳмқҙм–ҙлҘј м–№мқҖ кІғмқҙлӢӨ. scopeм—җ openidлҘј л„Јмңјл©ҙ access_tokenкіј н•Ёк»ҳ id_token(JWT)мқҙ л°ңкёүлҗңлӢӨ. id_tokenм—җлҠ” мӮ¬мҡ©мһҗ мӢқлі„ м •ліҙ(sub), мқёмҰқ мӢңк°Ғ(auth_time), мқёмҰқ л°©лІ•(amr) л“ұмқҙ л“Өм–ҙмһҲлӢӨ.

const crypto = require('crypto');

function generateAuthUrl() {
    // PKCE: Code Verifier / Code Challenge мғқм„ұ
    const codeVerifier = crypto.randomBytes(32).toString('base64url');
    const codeChallenge = crypto.createHash('sha256')
        .update(codeVerifier).digest('base64url');
    const state = crypto.randomBytes(16).toString('hex');  // CSRF л°©м–ҙ

    const params = new URLSearchParams({
        response_type: 'code',
        client_id:     process.env.OAUTH_CLIENT_ID,
        redirect_uri:  'https://myapp.com/callback',
        scope:         'openid profile email',
        state,
        code_challenge:        codeChallenge,
        code_challenge_method: 'S256'
    });

    // sessionм—җ м ҖмһҘ (callbackм—җм„ң кІҖмҰқ)
    session.codeVerifier = codeVerifier;
    session.state = state;

    return `https://accounts.google.com/o/oauth2/v2/auth?${params}`;
}

app.get('/callback', async (req, res) => {
    const { code, state } = req.query;

    // state кІҖмҰқ (CSRF л°©м–ҙ)

    const tokenResponse = await fetch('https://oauth2.googleapis.com/token', {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
        body: new URLSearchParams({
            grant_type:    'authorization_code',
            code,
            redirect_uri:  'https://myapp.com/callback',
            client_id:     process.env.OAUTH_CLIENT_ID,
            client_secret: process.env.OAUTH_CLIENT_SECRET,
            code_verifier: session.codeVerifier  // PKCE
        })
    });

    const tokens = await tokenResponse.json();
    // tokens = { access_token, refresh_token, id_token, expires_in }

    // мӮ¬мҡ©мһҗ м •ліҙ мЎ°нҡҢ
    const userInfo = await fetch('https://www.googleapis.com/oauth2/v2/userinfo', {
        headers: { Authorization: `Bearer ${tokens.access_token}` }
    }).then(r => r.json());

    // JWT id_token кІҖмҰқ
    const payload = await verifyIdToken(tokens.id_token);

let refreshPromise = null;

async function refreshAccessToken(refreshToken) {
    if (refreshPromise) return refreshPromise;

    refreshPromise = fetch('https://oauth2.googleapis.com/token', {
        method: 'POST',
        headers: { 'Content-Type': 'application/x-www-form-urlencoded' },
        body: new URLSearchParams({
            grant_type:    'refresh_token',
            refresh_token: refreshToken,
            client_id:     process.env.OAUTH_CLIENT_ID,
            client_secret: process.env.OAUTH_CLIENT_SECRET
        })
    })
    .then(r => r.json())
    .finally(() => { refreshPromise = null; });

    return refreshPromise;
}

        OAuth2UserInfo userInfo = OAuth2UserInfoFactory.of(registrationId, oAuth2User.getAttributes());
        User user = userRepo.findByEmail(userInfo.getEmail())

    }

    // JwtTokenProvider.java
@Component
public class JwtTokenProvider {
    @Value("${jwt.secret}") private String secret;
    private static final long ACCESS_TOKEN_EXPIRE  = 60 * 60 * 1000L;       // 1мӢңк°„
    private static final long REFRESH_TOKEN_EXPIRE = 7 * 24 * 60 * 60 * 1000L; // 7мқј

    private Key getKey() {
        return Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    }

    public String createAccessToken(Long userId, String role) {
        return Jwts.builder()
            .setSubject(userId.toString())
            .claim("role", role)
            .setIssuedAt(new Date())
            .setExpiration(new Date(System.currentTimeMillis() + ACCESS_TOKEN_EXPIRE))
            .signWith(getKey(), SignatureAlgorithm.HS256)
            .compact();
    }

    public Claims validateToken(String token) {
        try {
            return Jwts.parserBuilder()
                .setSigningKey(getKey()).build()
                .parseClaimsJws(token).getBody();
        } catch (ExpiredJwtException e) {
            throw new TokenExpiredException("нҶ нҒ°мқҙ л§ҢлЈҢлҗҳм—ҲмҠөлӢҲлӢӨ.");
        } catch (JwtException e) {
            throw new InvalidTokenException("мң нҡЁн•ҳм§Җ м•ҠмқҖ нҶ нҒ°мһ…лӢҲлӢӨ.");
        }
    }
}
