    1. лЎңл“ң л°ёлҹ°мӢұмқҳ кё°ліё к°ңл…җ¶

    1.1 лЎңл“ң л°ёлҹ°мӢұмқҙлһҖ?¶

    лЎңл“ң л°ёлҹ°мӢұ(Load Balancing)мқҖ л“Өм–ҙмҳӨлҠ” л„ӨнҠёмӣҢнҒ¬ нҠёлһҳн”Ҫмқ„ м—¬лҹ¬ м„ңлІ„лӮҳ м»ҙн“ЁнҢ… лҰ¬мҶҢмҠӨм—җ л¶„мӮ°мӢңмјң мІҳлҰ¬н•ҳлҠ” кё°мҲ мһ…лӢҲлӢӨ. мқҙлҠ” лӢЁмқј м„ңлІ„м—җ кіјл¶Җн•ҳк°Җ м§‘мӨ‘лҗҳлҠ” кІғмқ„ л°©м§Җн•ҳкі , м „мІҙ мӢңмҠӨн…ңмқҳ к°Җмҡ©м„ұкіј м„ұлҠҘмқ„ н–ҘмғҒмӢңнӮӨлҠ” н•өмӢ¬м Ғмқё м—ӯн• мқ„ н•©лӢҲлӢӨ.

    1.2 лЎңл“ң л°ёлҹ°мӢұмқҳ н•өмӢ¬ лӘ©м Ғ¶

    1.2.1 нҠёлһҳн”Ҫ л¶„мӮ°¶

    • к· л“ұ л¶„мӮ°: лӘЁл“  м„ңлІ„к°Җ лҸҷмқјн•ң м–‘мқҳ нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳлҸ„лЎқ л¶„л°°
    • к°ҖмӨ‘м№ҳ кё°л°ҳ л¶„мӮ°: м„ңлІ„мқҳ м„ұлҠҘмқҙлӮҳ мҡ©лҹүм—җ л”°лқј м°Ёл“ұм ҒмңјлЎң нҠёлһҳн”Ҫ н• лӢ№
    • м§ҖлҰ¬м Ғ л¶„мӮ°: мӮ¬мҡ©мһҗмқҳ мң„м№ҳм—җ л”°лқј к°ҖмһҘ к°Җк№Ңмҡҙ м„ңлІ„лЎң лқјмҡ°нҢ…

    1.2.2 кі к°Җмҡ©м„ұ нҷ•ліҙ¶

    • мһҘм•  кІ©лҰ¬: н•ң м„ңлІ„м—җ л¬ём ңк°Җ л°ңмғқн•ҙлҸ„ лӢӨлҘё м„ңлІ„л“Өмқҙ м„ңл№„мҠӨ м§ҖмҶҚ
    • мһҗлҸҷ ліөкө¬: мһҘм•  м„ңлІ„лҘј мһҗлҸҷмңјлЎң к°җм§Җн•ҳкі  нҠёлһҳн”Ҫм—җм„ң м ңмҷё
    • л¬ҙмӨ‘лӢЁ м„ңл№„мҠӨ: м„ңлІ„ м¶”к°Җ/м ңкұ° мӢңм—җлҸ„ м„ңл№„мҠӨ мӨ‘лӢЁ м—Ҷмқҙ мҡҙмҳҒ

    1.2.3 м„ұлҠҘ мөңм Ғнҷ”¶

    • мқ‘лӢө мӢңк°„ лӢЁм¶•: к°ҖмһҘ л№ лҘё м„ңлІ„лЎң мҡ”мІӯ лқјмҡ°нҢ…
    • мІҳлҰ¬лҹү мҰқлҢҖ: м—¬лҹ¬ м„ңлІ„мқҳ лі‘л ¬ мІҳлҰ¬лЎң м „мІҙ мІҳлҰ¬лҹү н–ҘмғҒ
    • лҰ¬мҶҢмҠӨ нҡЁмңЁм„ұ: м„ңлІ„ лҰ¬мҶҢмҠӨмқҳ к· л“ұн•ң нҷңмҡ©

    1.3 OSI 7кі„мёөкіј лЎңл“ң л°ёлҹ°мӢұ¶

    лЎңл“ң л°ёлҹ°м„ңлҠ” OSI 7кі„мёө лӘЁлҚёмқҳ м–ҙлҠҗ кі„мёөм—җм„ң лҸҷмһ‘н•ҳлҠҗлғҗм—җ л”°лқј к·ё нҠ№м„ұкіј кё°лҠҘмқҙ лӢ¬лқјм§‘лӢҲлӢӨ:

    1.3.1 4кі„мёө (Transport Layer) лЎңл“ң л°ёлҹ°мӢұ¶

    • нҠ№м§•: TCP/UDP нҢЁнӮ·мқҳ IP мЈјмҶҢмҷҖ нҸ¬нҠё лІҲнҳёл§Ңмқ„ кё°л°ҳмңјлЎң л¶„мӮ°
    • мһҘм җ: л№ лҘё мІҳлҰ¬ мҶҚлҸ„, лӮ®мқҖ м§Җм—°мӢңк°„, лӢЁмҲңн•ң кө¬мЎ°
    • лӢЁм җ: м• н”ҢлҰ¬мјҖмқҙм…ҳ л ҲлІЁмқҳ м„ёл°Җн•ң м ңм–ҙ л¶Ҳк°ҖлҠҘ
    • мӮ¬мҡ© мӮ¬лЎҖ: лҚ°мқҙн„°лІ мқҙмҠӨ м—°кІ°, кІҢмһ„ м„ңлІ„, мӢӨмӢңк°„ нҶөмӢ 

    1.3.2 7кі„мёө (Application Layer) лЎңл“ң л°ёлҹ°мӢұ¶

    • нҠ№м§•: HTTP/HTTPS мҡ”мІӯмқҳ лӮҙмҡ©мқ„ л¶„м„қн•ҳм—¬ лқјмҡ°нҢ… кІ°м •
    • мһҘм җ: м„ёл°Җн•ң лқјмҡ°нҢ… м ңм–ҙ, мҪҳн…җмё  кё°л°ҳ л¶„мӮ°, ліҙм•Ҳ кё°лҠҘ
    • лӢЁм җ: мғҒлҢҖм ҒмңјлЎң лҶ’мқҖ м§Җм—°мӢңк°„, ліөмһЎн•ң кө¬мЎ°
    • мӮ¬мҡ© мӮ¬лЎҖ: мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳ, API м„ңлІ„, л§ҲмқҙнҒ¬лЎңм„ңл№„мҠӨ

    2. AWS лЎңл“ң л°ёлҹ°м„ң к°ңмҡ”¶

    2.1 AWS Elastic Load Balancing (ELB) м„ңл№„мҠӨ¶

    AWSмқҳ Elastic Load BalancingмқҖ нҒҙлқјмҡ°л“ң нҷҳкІҪм—җм„ң м ңкіөлҗҳлҠ” мҷ„м „ кҙҖлҰ¬нҳ• лЎңл“ң л°ёлҹ°мӢұ м„ңл№„мҠӨмһ…лӢҲлӢӨ. мқҙ м„ңл№„мҠӨлҠ” н•ҳл“ңмӣЁм–ҙ м„Өм№ҳлӮҳ мҶҢн”„нҠёмӣЁм–ҙ кө¬м„ұ м—ҶмқҙлҸ„ мҰүмӢң мӮ¬мҡ©н•  мҲҳ мһҲмңјл©°, мһҗлҸҷмңјлЎң нҷ•мһҘлҗҳкі  кі к°Җмҡ©м„ұмқ„ м ңкіөн•©лӢҲлӢӨ.

    2.2 AWS лЎңл“ң л°ёлҹ°м„ңмқҳ н•өмӢ¬ нҠ№м§•¶

    2.2.1 мҷ„м „ кҙҖлҰ¬нҳ• м„ңл№„мҠӨ¶

    • мһҗлҸҷ нҷ•мһҘ: нҠёлһҳн”Ҫ мҰқк°Җм—җ л”°лқј мһҗлҸҷмңјлЎң мҡ©лҹү нҷ•мһҘ
    • мһҗлҸҷ ліөкө¬: мһҘм•  л°ңмғқ мӢң мһҗлҸҷмңјлЎң лҢҖмІҙ лҰ¬мҶҢмҠӨ н• лӢ№
    • л¬ҙмӨ‘лӢЁ мҡҙмҳҒ: AWSк°Җ л°ұк·ёлқјмҡҙл“ңм—җм„ң лӘЁл“  кҙҖлҰ¬ мһ‘м—… мҲҳн–ү

    2.2.2 лӢӨмӨ‘ к°Җмҡ©мҳҒм—ӯ м§Җмӣҗ¶

    • м§Җм—ӯ л¶„мӮ°: м—¬лҹ¬ к°Җмҡ©мҳҒм—ӯм—җ кұёміҗ лЎңл“ң л°ёлҹ°м„ң мқёмҠӨн„ҙмҠӨ л°°м№ҳ
    • мһҘм•  кІ©лҰ¬: н•ң к°Җмҡ©мҳҒм—ӯ мһҘм•  мӢңм—җлҸ„ лӢӨлҘё мҳҒм—ӯм—җм„ң м„ңл№„мҠӨ м§ҖмҶҚ
    • м§Җм—°мӢңк°„ мөңм Ғнҷ”: мӮ¬мҡ©мһҗмҷҖ к°ҖмһҘ к°Җк№Ңмҡҙ к°Җмҡ©мҳҒм—ӯмңјлЎң нҠёлһҳн”Ҫ лқјмҡ°нҢ…

    2.2.3 ліҙм•Ҳ нҶөн•©¶

    • AWS WAF нҶөн•©: мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳ л°©нҷ”лІҪкіјмқҳ мҷ„лІҪн•ң м—°лҸҷ
    • SSL/TLS мў…лЈҢ: мқёмҰқм„ң кҙҖлҰ¬мҷҖ м•”нҳёнҷ”/ліөнҳёнҷ” мІҳлҰ¬
    • VPC ліҙм•Ҳ: к°ҖмғҒ мӮ¬м„Ө нҒҙлқјмҡ°л“ң лӮҙм—җм„ңмқҳ м•Ҳм „н•ң нҶөмӢ 

    2.3 AWS лЎңл“ң л°ёлҹ°м„ң мң нҳ•¶

    AWSлҠ” нҳ„мһ¬ 4к°Җм§Җ мң нҳ•мқҳ лЎңл“ң л°ёлҹ°м„ңлҘј м ңкіөн•©лӢҲлӢӨ:

    1. ALB (Application Load Balancer) - 7кі„мёө HTTP/HTTPS лЎңл“ң л°ёлҹ°м„ң
    2. NLB (Network Load Balancer) - 4кі„мёө TCP/UDP лЎңл“ң л°ёлҹ°м„ң
    3. GWLB (Gateway Load Balancer) - 3кі„мёө кІҢмқҙнҠёмӣЁмқҙ лЎңл“ң л°ёлҹ°м„ң
    4. CLB (Classic Load Balancer) - л Ҳкұ°мӢң лЎңл“ң л°ёлҹ°м„ң (кө¬ ELB)

    2.4 AWS ELB мң нҳ• к°„лӢЁ л№„көҗ¶

    лҢҖл¶Җл¶„мқҳ мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳм—җм„ңлҠ” ALBлҘј, кі м„ұлҠҘ TCP/UDP мІҳлҰ¬к°Җ н•„мҡ”н•ң кІҪмҡ° NLBлҘј мӮ¬мҡ©н•©лӢҲлӢӨ.

    3. ALB (Application Load Balancer)¶

    3.1 ALBмқҳ кё°ліё к°ңл…җ¶

    Application Load BalancerлҠ” OSI 7кі„мёөмқҳ Application Layerм—җм„ң лҸҷмһ‘н•ҳлҠ” кі кёү лЎңл“ң л°ёлҹ°м„ңмһ…лӢҲлӢӨ. HTTP/HTTPS нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳлҠ”лҚ° нҠ№нҷ”лҗҳм–ҙ мһҲмңјл©°, нҳ„лҢҖм Ғмқё мӣ№ м• н”ҢлҰ¬мјҖмқҙм…ҳкіј л§ҲмқҙнҒ¬лЎңм„ңл№„мҠӨ м•„нӮӨн…ҚмІҳм—җ мөңм Ғнҷ”лҗҳм–ҙ мһҲмҠөлӢҲлӢӨ.

    3.2 ALBмқҳ н•өмӢ¬ м•„нӮӨн…ҚмІҳ¶

    3.2.1 7кі„мёө мІҳлҰ¬ л°©мӢқ¶

    ALBлҠ” HTTP/HTTPS мҡ”мІӯмқҳ м„ёл¶Җ лӮҙмҡ©мқ„ л¶„м„қн•ҳм—¬ лқјмҡ°нҢ… кІ°м •мқ„ лӮҙлҰҪлӢҲлӢӨ:

    • HTTP н—ӨлҚ” л¶„м„қ: Host, User-Agent, Authorization л“ұмқҳ н—ӨлҚ” м •ліҙ нҷңмҡ©
    • URL кІҪлЎң л¶„м„қ: мҡ”мІӯ кІҪлЎңлҘј кё°л°ҳмңјлЎң н•ң м„ёл°Җн•ң лқјмҡ°нҢ…
    • HTTP л©”м„ңл“ң л¶„м„қ: GET, POST, PUT л“ұ л©”м„ңл“ңлі„ лқјмҡ°нҢ…
    • мҝјлҰ¬ нҢҢлқјлҜён„° л¶„м„қ: URLмқҳ мҝјлҰ¬ мҠӨнҠёл§Ғмқ„ нҶөн•ң мЎ°кұҙл¶Җ лқјмҡ°нҢ…

    3.2.2 нғҖкІҹ к·ёлЈ№ кё°л°ҳ м•„нӮӨн…ҚмІҳ¶

    ALBлҠ” нғҖкІҹ к·ёлЈ№(Target Group)мқҙлқјлҠ” к°ңл…җмқ„ лҸ„мһ…н•ҳм—¬ мң м—°н•ң нҠёлһҳн”Ҫ л¶„мӮ°мқ„ м ңкіөн•©лӢҲлӢӨ:

    • лӢӨмӨ‘ нғҖкІҹ к·ёлЈ№: н•ҳлӮҳмқҳ ALBк°Җ м—¬лҹ¬ нғҖкІҹ к·ёлЈ№мқ„ кҙҖлҰ¬
    • лҸҷм Ғ нғҖкІҹ кҙҖлҰ¬: нғҖкІҹмқҳ м¶”к°Җ/м ңкұ°к°Җ мӢӨмӢңк°„мңјлЎң л°ҳмҳҒ
    • нғҖкІҹлі„ н—¬мҠӨ мІҙнҒ¬: к°Ғ нғҖкІҹ к·ёлЈ№л§ҲлӢӨ лҸ…лҰҪм Ғмқё н—¬мҠӨ мІҙнҒ¬ м„Өм •

    4. NLB (Network Load Balancer)¶

    4.1 NLBмқҳ кё°ліё к°ңл…җ¶

    Network Load BalancerлҠ” OSI 7кі„мёөмқҳ Transport Layer(4кі„мёө)м—җм„ң лҸҷмһ‘н•ҳлҠ” кі м„ұлҠҘ лЎңл“ң л°ёлҹ°м„ңмһ…лӢҲлӢӨ. TCP, UDP, TLS нҠёлһҳн”Ҫмқ„ мІҳлҰ¬н•ҳл©°, мҙҲкі м„ұлҠҘкіј мҙҲм Җм§Җм—°мӢңк°„мқ„ м ңкіөн•ҳлҠ” кІғмқҙ н•өмӢ¬ нҠ№м§•мһ…лӢҲлӢӨ.

    4.2 NLBмқҳ м•„нӮӨн…ҚмІҳ нҠ№м§•¶

    4.2.1 4кі„мёө мІҳлҰ¬ л°©мӢқ¶

    NLBлҠ” нҢЁнӮ·мқҳ IP мЈјмҶҢмҷҖ нҸ¬нҠё лІҲнҳёл§Ңмқ„ кё°л°ҳмңјлЎң лқјмҡ°нҢ… кІ°м •мқ„ лӮҙлҰҪлӢҲлӢӨ:

    • нҢЁнӮ· л ҲлІЁ мІҳлҰ¬: м• н”ҢлҰ¬мјҖмқҙм…ҳ лӮҙмҡ©мқ„ л¶„м„қн•ҳм§Җ м•Ҡкі  нҢЁнӮ· н—ӨлҚ”л§Ң нҷ•мқё
    • м—°кІ° мң м§Җ: TCP м—°кІ°мқҳ нҠ№м„ұмқ„ нҷңмҡ©н•ң нҡЁмңЁм Ғмқё нҠёлһҳн”Ҫ л¶„мӮ°
    • н”„лЎңнҶ мҪң нҲ¬лӘ…м„ұ: мғҒмң„ кі„мёө н”„лЎңнҶ мҪңм—җ л¬ҙкҙҖн•ҳкІҢ лҸҷмһ‘

    4.2.2 кі м • IP мЈјмҶҢ м§Җмӣҗ¶

    NLBмқҳ к°ҖмһҘ мӨ‘мҡ”н•ң нҠ№м§• мӨ‘ н•ҳлӮҳлҠ” кі м • IP мЈјмҶҢлҘј м ңкіөн•ңлӢӨлҠ” кІғмһ…лӢҲлӢӨ:

    • Elastic IP н• лӢ№: к°Ғ к°Җмҡ©мҳҒм—ӯл§ҲлӢӨ кі м • IP мЈјмҶҢ н• лӢ№ к°ҖлҠҘ
    • нҒҙлқјмқҙм–ёнҠё IP ліҙмЎҙ: мӣҗліё нҒҙлқјмқҙм–ёнҠё IP мЈјмҶҢлҘј л°ұм—”л“ңлЎң м „лӢ¬
    • DNS кё°л°ҳ м ‘к·ј: кі м • IPлҘј DNSм—җ л“ұлЎқн•ҳм—¬ м•Ҳм •м Ғмқё м ‘к·ј м ңкіө

    5. ELB (Elastic Load Balancer)¶

    5.1 ELBмқҳ кё°ліё к°ңл…җ¶

    Elastic Load Balancer (ELB)лҠ” AWSмқҳ л Ҳкұ°мӢң лЎңл“ң л°ёлҹ°м„ңмһ…лӢҲлӢӨ. мқҙм „м—җлҠ” Classic Load Balancer (CLB)лқјкі  л¶Ҳл ёмңјл©°, AWSк°Җ м ңкіөн•ң мөңмҙҲмқҳ лЎңл“ң л°ёлҹ°м„ң м„ңл№„мҠӨмһ…лӢҲлӢӨ. нҳ„мһ¬лҠ” мғҲлЎңмҡҙ н”„лЎңм қнҠём—җм„ңлҠ” к¶ҢмһҘлҗҳм§Җ м•Ҡм§Җл§Ң, кё°мЎҙ мӢңмҠӨн…ңкіјмқҳ нҳёнҷҳм„ұмқ„ мң„н•ҙ м—¬м „нһҲ м ңкіөлҗҳкі  мһҲмҠөлӢҲлӢӨ.

    к°ңм„  л°©м•Ҳ: - к°ҖлҠҘн•ң кІҪмҡ° HTTP/HTTPS н—¬мҠӨ мІҙнҒ¬лЎң м „нҷҳ - м• н”ҢлҰ¬мјҖмқҙм…ҳ л ҲлІЁм—җм„ңмқҳ мғҒнғң лӘЁлӢҲн„°л§Ғ лі‘н–ү - мҷёл¶Җ лӘЁлӢҲн„°л§Ғ лҸ„кө¬мҷҖмқҳ м—°лҸҷ

    8.3 лӘЁлӢҲн„°л§Ғ л°Ҹ лЎңк№…¶

    8.3.1 CloudWatch м§Җн‘ң нҷңмҡ©¶

    мЈјмҡ” лӘЁлӢҲн„°л§Ғ м§Җн‘ң:

    ALB м§Җн‘ң: - RequestCount: мІҳлҰ¬лҗң мҡ”мІӯ мҲҳ - TargetResponseTime: нғҖкІҹ мқ‘лӢө мӢңк°„ - HTTPCode_Target_2XX_Count: м„ұкіө мқ‘лӢө мҲҳ - HTTPCode_Target_4XX_Count: нҒҙлқјмқҙм–ёнҠё мҳӨлҘҳ мҲҳ - HTTPCode_Target_5XX_Count: м„ңлІ„ мҳӨлҘҳ мҲҳ

    NLB м§Җн‘ң: - ActiveFlowCount: нҷңм„ұ м—°кІ° мҲҳ - NewFlowCount: мғҲ м—°кІ° мҲҳ - ProcessedBytes: мІҳлҰ¬лҗң л°”мқҙнҠё мҲҳ - TCP_Target_Reset_Count: TCP лҰ¬м…Ӣ мҲҳ

    8.3.2 м•Ўм„ёмҠӨ лЎңк·ё л¶„м„қ¶

    лЎңк·ё кё°л°ҳ м•ҢлҰј м„Өм •: - 5xx мҳӨлҘҳмңЁ мһ„кі„к°’ мҙҲкіј мӢң м•ҢлҰј - мқ‘лӢө мӢңк°„ мһ„кі„к°’ мҙҲкіј мӢң м•ҢлҰј - л№„м •мғҒм Ғмқё нҠёлһҳн”Ҫ нҢЁн„ҙ к°җм§Җ мӢң м•ҢлҰј

    8.4 ліҙм•Ҳ кі л ӨмӮ¬н•ӯ¶

    ліҙм•Ҳ лӘЁлІ” мӮ¬лЎҖ: - мөңмӢ  TLS лІ„м „ мӮ¬мҡ© (TLS 1.2 мқҙмғҒ) - к°•л Ҙн•ң м•”нҳёнҷ” мҠӨмң„нҠё мӮ¬мҡ© - HSTS (HTTP Strict Transport Security) н—ӨлҚ” м„Өм • - м •кё°м Ғмқё ліҙм•Ҳ м—…лҚ°мқҙнҠё м Ғмҡ©
