    x-quorum-initial-group-sizeΈΛΦ ΊΒ┤ΈθυΉΛνΊΕ░ ΈΖ╕ΈΥε ΉΙα Ήζ┤ΊΧαΈκε ΉΕνΉιΧΊΧ┤ΉΧ╝ ΊΧεΈΜν. 3ΈΖ╕ΈΥε ΊΒ┤ΈθυΉΛνΊΕ░Έσ┤ 3Ήζ┤ ΉιΒΈΜ╣ΊΧαΈΜν. 5ΈΖ╕ΈΥε ΊΒ┤ΈθυΉΛνΊΕ░ΉΩΡΉΕε 5Έκε ΉΕνΉιΧΊΧαΈσ┤ ΈςρΈΥι ΈΖ╕ΈΥεΉΩΡ Έ│╡ΉιεΈΡαΉπΑΈπΝ ΉΥ░Ά╕░ ΉΕ▒ΈΛξΉζ┤ ΈΨρΉΨ┤ΉπΕΈΜν.

    Quorum QueueΉΩΡΉΕε Ήμ╝ΉζαΊΧι ΉιΡ:

    • exclusive, non-durable Ήα╡ΉΖαΉζΑ ΉΥ╕ ΉΙα ΉΩΗΈΜν. ΊΧφΉΔΒ durableΉζ┤ΈΜν.
    • ΈσΦΉΜεΉπΑΆ░Α requeueΈΡαΈσ┤ ΉδΡΈηα ΉΙεΉΕεΆ░Α Έ│┤ΉηξΈΡαΉπΑ ΉΧΛΈΛΦΈΜν. ΉΙεΉΕεΆ░Α ΉνΣΉγΦΊΧε Ή▓αΈουΉΩΡΉΕε nack+requeueΈξ╝ ΉΥ╕ ΈΧΝ Ήμ╝ΉζαΊΧ┤ΉΧ╝ ΊΧεΈΜν.
    • poison message handlingΉζΕ ΉεΕΊΧ┤ x-delivery-limitΉζΕ ΉΕνΉιΧΊΧι ΉΙα ΉηΙΈΜν. Ήζ┤ ΊγθΉΙαΈξ╝ Ή┤ΙΆ│╝ΊΧαΈσ┤ DLQΈκε Ά░ΕΈΜν.
    args.put("x-delivery-limit", 5); // 5Έ▓Ι ΉηυΉιΕΈΜυ ΊδΕ DLQΈκε

    Lazy Queue¶

    Lazy QueueΈΛΦ ΈσΦΉΜεΉπΑΈξ╝ Ή╡εΈΝΑΊΧε ΈΦΦΉΛνΊΒυΉΩΡ ΉιΑΉηξΊΧαΆ│ι ConsumerΆ░Α ΉγΦΉ▓φΊΧι ΈΧΝΈπΝ ΈσΦΈςρΈουΈκε ΉαυΈο░ΈΜν.

    Map<String, Object> args = new HashMap<>();
args.put("x-queue-mode", "lazy");
channel.queueDeclare("my.lazy.queue", true, false, false, args);

    Ήζ╝Έ░α ΊΒΡΈΛΦ ΈσΦΉΜεΉπΑΈξ╝ ΈσΦΈςρΈουΉΩΡ ΉειΉπΑΊΧαΈΜνΆ░Α ΈσΦΈςρΈου pressureΆ░Α ΉΔζΆ╕░Έσ┤ ΈΦΦΉΛνΊΒυΈκε ΈΓ┤Έο░ΈΜν. Lazy QueueΈΛΦ Ή▓αΉζΝΈ╢ΑΊΕ░ ΈΦΦΉΛνΊΒυΉΩΡ ΉΥ┤ΈΜν.

    ΉΥ░ΈΛΦ Ά▓╜Ήγ░:

    • ΊΒΡΉΩΡ ΈσΦΉΜεΉπΑΆ░Α ΉΙαΈ░▒ΈπΝ Ά▒┤ ΉΝΥΉζ┤ΈΛΦ Ά╡υΉκ░ (ConsumerΆ░Α Ά░ΕΊΩΡΉιΒΉε╝Έκε Ή▓αΈου)
    • Memory AlarmΉζ┤ ΉηΡΉμ╝ Έ░εΉΔζΊΧαΈΛΦ ΊβαΆ▓╜
    • Ή▓αΈουΈθΚΈ│┤ΈΜν ΉΧΙΉιΧΉΕ▒Ήζ┤ ΉνΣΉγΦΊΧε Ά▓╜Ήγ░

    RabbitMQ 3.12Έ╢ΑΊΕ░ Quorum QueueΈΛΦ Ά╕░Έ│╕ΉιΒΉε╝Έκε lazy ΈΠβΉηΣΉζΕ ΊΧεΈΜν. Classic QueueΉΩΡΉΕεΈπΝ Έ│ΕΈΠΕ ΉΕνΉιΧΉζ┤ ΊΧΕΉγΦΊΧαΈΜν.

    Priority Queue¶

    Map<String, Object> args = new HashMap<>();
args.put("x-max-priority", 10); // Ήγ░ΉΕιΉΙεΉεΕ Έ▓ΦΉεΕ 0~10
channel.queueDeclare("my.priority.queue", true, false, false, args);

    ΈσΦΉΜεΉπΑ Έ░εΊΨΚ ΉΜε priorityΈξ╝ ΉπΑΉιΧΊΧεΈΜν:

    AMQP.BasicProperties props = new AMQP.BasicProperties.Builder()
    .priority(5)
    .build();
channel.basicPublish("exchange", "routing.key", props, body);

    x-max-priority Ά░ΤΉζ┤ ΊΒ┤ΉΙαΈκζ ΈΓ┤Έ╢ΑΉιΒΉε╝Έκε Ήγ░ΉΕιΉΙεΉεΕΈ│Ε ΉΕεΈ╕ΝΊΒΡΈξ╝ ΈπΟΉζ┤ ΈπΝΈΥιΈΜν. 10 Ήζ┤ΊΧαΈκε ΉΕνΉιΧΊΧαΈΛΦ Ά▓Ν ΉλΜΈΜν. 255Ά╣ΝΉπΑ Ά░ΑΈΛξΊΧαΉπΑΈπΝ ΈσΦΈςρΈουΉβΑ CPU ΉανΈ▓ΕΊΩνΈΥεΆ░Α ΉΜυΊΧαΈΜν.

    priorityΆ░Α Ά░βΉζΑ ΈσΦΉΜεΉπΑΈΒ╝ΈουΈΛΦ FIFO ΉΙεΉΕεΈξ╝ ΉειΉπΑΊΧεΈΜν. ConsumerΉζα prefetchΆ░Α ΈΗΤΉε╝Έσ┤ Έ╕ΝΈκεΉ╗ν Ή╕κΉΩΡΉΕε ΉιΧΈιυΊΧ┤ΈΠΕ Consumer Ή╕κ Έ▓ΕΊΞ╝ΉΩΡΉΕε ΉΙεΉΕεΆ░Α ΉΕηΉζ╝ ΉΙα ΉηΙΈΜν. Ήγ░ΉΕιΉΙεΉεΕΆ░Α ΉνΣΉγΦΊΧαΈσ┤ prefetchΈξ╝ 1Έκε ΈΓχΉ╢ΦΈΛΦ Ά▓Ν ΉΧΙΉιΕΊΧαΈΜν.

    Memory/Disk AlarmΆ│╝ Flow Control¶

    Memory Alarm¶

    RabbitMQΈΛΦ ΉΓυΉγσ ΈσΦΈςρΈουΆ░Α ΉηΕΆ│ΕΆ░ΤΉζΕ Ή┤ΙΆ│╝ΊΧαΈσ┤ ΈςρΈΥι PublisherΉζα ConnectionΉζΕ Έ╕ΦΈκεΊΓ╣ΊΧεΈΜν. ConsumerΈΛΦ ΉιΧΉΔΒ ΈΠβΉηΣΊΧεΈΜν.

    # rabbitmq.conf
vm_memory_high_watermark.relative = 0.4  # ΉΜεΉΛνΊΖε ΈσΦΈςρΈουΉζα 40% (Ά╕░Έ│╕Ά░Τ)
# ΈαΡΈΛΦ ΉιΙΈΝΑΆ░Τ
# vm_memory_high_watermark.absolute = 2GB

    Memory AlarmΉζ┤ Έ░εΉΔζΊΧαΈσ┤:

    1. Publisher ConnectionΉζ┤ Έ╕ΦΈκεΊΓ╣ΈΡεΈΜν (write Έ▓ΕΊΞ╝Ά░Α Ά╜Κ Ή░υ Ά▓ΔΉ▓αΈθ╝ ΈΠβΉηΣ)
    2. Έ╕ΝΈκεΉ╗νΆ░Α ΈσΦΈςρΈουΈξ╝ ΊβΧΈ│┤ΊΧαΈινΆ│ι ΊΒΡΉζα ΈσΦΉΜεΉπΑΈξ╝ ΈΦΦΉΛνΊΒυΈκε ΈΓ┤Έο░ΈΜν (paging)
    3. ΈσΦΈςρΈουΆ░Α ΉηΕΆ│ΕΆ░Τ ΉΧΕΈηαΈκε ΈΓ┤ΈινΆ░ΑΈσ┤ Έ╕ΦΈκεΊΓ╣Ήζ┤ ΊΤΑΈο░ΈΜν

    vm_memory_high_watermark_paging_ratio(Ά╕░Έ│╕ 0.5)Έξ╝ ΉΕνΉιΧΊΧαΈσ┤ ΉηΕΆ│ΕΆ░ΤΉζα 50%ΉΩΡΉΕε Έψ╕Έου pagingΉζΕ ΉΜεΉηΣΊΧεΈΜν.

    Disk Alarm¶

    disk_free_limit.relative = 1.5  # ΈσΦΈςρΈου ΊΒυΆ╕░Ήζα 1.5Έ░░ (Ά╕░Έ│╕Ά░Τ)
# ΈαΡΈΛΦ ΉιΙΈΝΑΆ░Τ
disk_free_limit.absolute = 5GB

    ΈΦΦΉΛνΊΒυ ΉΩυΉει Ά│╡Ά░ΕΉζ┤ ΉηΕΆ│ΕΆ░Τ ΉΧΕΈηαΈκε ΈΨρΉΨ┤ΉπΑΈσ┤ Memory AlarmΆ│╝ ΈπΙΉ░υΆ░ΑΉπΑΈκε PublisherΈξ╝ Έ╕ΦΈκεΊΓ╣ΊΧεΈΜν. ΉΩυΆ╕░ΉΩΡ ΈΞΦΊΧ┤ ΈσΦΉΜεΉπΑΈξ╝ ΈΦΦΉΛνΊΒυΉΩΡ ΉΥ░ΈΛΦ Ά▓ΔΈΠΕ ΉνΣΈΜρΊΧεΈΜν.

    ΈΦΦΉΛνΊΒυΆ░Α Ά░ΑΈΥζ Ή░ρΈσ┤ Έ╕ΝΈκεΉ╗ν ΉηΡΉ▓┤Ά░Α Ά╕░ΈΠβ Έ╢ΙΆ░Α ΉΔΒΊΔεΉΩΡ Έ╣ιΉπΙ ΉΙα ΉηΙΈΜν. Ήγ┤ΉαΒΉΩΡΉΕε ΈΦΦΉΛνΊΒυ ΈςρΈΜΙΊΕ░ΈπΒΉζΑ ΊΧΕΉΙαΈΜν.

    Flow Control¶

    PublisherΆ░Α Έ╕ΝΈκεΉ╗ν Ή▓αΈου ΉΗΞΈΠΕΈ│┤ΈΜν Έ╣ιΈξ┤Ά▓Ν ΈσΦΉΜεΉπΑΈξ╝ Έ│┤ΈΓ┤Έσ┤ ΈΓ┤Έ╢ΑΉιΒΉε╝Έκε credit Ά╕░Έ░α flow controlΉζ┤ ΈΠβΉηΣΊΧεΈΜν.

    Publisher έΗΤ [TCP Buffer] έΗΤ [Connection Process] έΗΤ [Channel Process] έΗΤ [Queue Process]

    Ά░Β ΈΜρΆ│ΕΉζα ΊΦΕΈκεΉΕ╕ΉΛνΆ░Α ΈΤνΉς╜ ΊΦΕΈκεΉΕ╕ΉΛνΉΩΡΆ▓Ν creditΉζΕ Έ░εΆ╕ΚΊΧεΈΜν. creditΉζ┤ ΉΗΝΉπΕΈΡαΈσ┤ ΉΧηΉς╜ ΊΦΕΈκεΉΕ╕ΉΛνΆ░Α Ήζ╝ΉΜε ΉιΧΉπΑΊΧεΈΜν. Management UIΉΩΡΉΕε Connection ΉΔΒΊΔεΆ░Α flowΈκε ΊΣεΉΜεΈΡαΈσ┤ ΊΧ┤ΈΜ╣ ConnectionΉΩΡ flow controlΉζ┤ Ά▒╕Έο░ Ά▓ΔΉζ┤ΈΜν.

    flow controlΉζ┤ ΉηΡΉμ╝ Έ░εΉΔζΊΧαΈσ┤:

    • Publisher ΉΗΞΈΠΕΈξ╝ ΉνΕΉζ┤Ά▒░ΈΓα
    • ConsumerΈξ╝ ΈΛαΈουΆ▒░ΈΓα
    • ΊΒΡΈξ╝ ΉΩυΈθυ Ά░εΈκε Έ╢ΕΉΓ░(consistent hash exchange ΈΥ▒)ΊΧαΆ▒░ΈΓα
    • ΈΖ╕ΈΥεΈξ╝ Ή╢ΦΆ░ΑΊΧ┤ΉΧ╝ ΊΧεΈΜν

    ShovelΆ│╝ Federation¶

    ΈΣΡ Ά╕░ΈΛξ ΈςρΈΣΡ ΉΕεΈκε ΈΜνΈξ╕ RabbitMQ ΊΒ┤ΈθυΉΛνΊΕ░(ΈαΡΈΛΦ ΈΖ╕ΈΥε) Ά░ΕΉΩΡ ΈσΦΉΜεΉπΑΈξ╝ ΉιΕΈΜυΊΧαΈΛΦ ΉγσΈΠΕΈΜν.

    Shovel¶

    ΊΧεΉς╜ ΊΒΡΉΩΡΉΕε ΈσΦΉΜεΉπΑΈξ╝ Ά║╝ΈΓ┤ΉΕε ΈΜνΈξ╕ Ής╜ exchange/queueΈκε ΈΕμΈΛΦΈΜν. ΈΜρΉΙεΊΧε ΊΠυΉζ╕ΊΛ╕-ΊΙυ-ΊΠυΉζ╕ΊΛ╕ ΉιΕΈΜυΉζ┤ΈΜν.

    rabbitmq-plugins enable rabbitmq_shovel
rabbitmq-plugins enable rabbitmq_shovel_management  # Management UI ΉΩ░ΈΠβ

    Dynamic Shovel (APIΈκε ΉΕνΉιΧ):

    rabbitmqctl set_parameter shovel my-shovel \
  '{"src-protocol": "amqp091", "src-uri": "amqp://user:pass@source-host",
    "src-queue": "source-queue",
    "dest-protocol": "amqp091", "dest-uri": "amqp://user:pass@dest-host",
    "dest-exchange": "dest-exchange",
    "ack-mode": "on-confirm",
    "reconnect-delay": 5}'

    ack-mode Ήα╡ΉΖα:

    • on-confirm: ΈΝΑΉΔΒ Έ╕ΝΈκεΉ╗νΉζα confirmΉζΕ Έ░δΉζΑ ΊδΕ ΉδΡΈ│╕ΉΩΡΉΕε ack. ΈσΦΉΜεΉπΑ ΉειΉΜν ΉΩΗΉζΝ.
    • on-publish: ΈΝΑΉΔΒ Έ╕ΝΈκεΉ╗νΉΩΡ publishΊΧε ΉοΚΉΜε ΉδΡΈ│╕ΉΩΡΉΕε ack. confirmΈ│┤ΈΜν Έ╣ιΈξ┤ΉπΑΈπΝ ΉειΉΜν Ά░ΑΈΛξ.
    • no-ack: ΉδΡΈ│╕ΉΩΡΉΕε auto-ack. Ά░ΑΉηξ Έ╣ιΈξ┤ΉπΑΈπΝ ΉειΉΜν ΉεΕΊΩα ΊΒ╝.

    Federation¶

    ExchangeΈΓα QueueΈξ╝ upstream/downstream Ά┤ΑΆ│ΕΈκε ΉΩ░Ά▓░ΊΧεΈΜν. ShovelΆ│╝ ΈΜυΈου "Ά╡υΈΠΖ" Ά╕░Έ░αΉζ┤Έζ╝ΉΕε downstream Ής╜ΉΩΡΉΕε ConsumerΆ░Α ΉηΙΉζΕ ΈΧΝΈπΝ ΈσΦΉΜεΉπΑΈξ╝ Ά░ΑΉι╕ΉαρΈΜν.

    rabbitmq-plugins enable rabbitmq_federation
rabbitmq-plugins enable rabbitmq_federation_management

    # upstream ΉιΧΉζα
rabbitmqctl set_parameter federation-upstream my-upstream \
  '{"uri": "amqp://user:pass@upstream-host", "expires": 3600000}'

# policyΈκε exchangeΉΩΡ federation ΉιΒΉγσ
rabbitmqctl set_policy federate-me "^federated\." \
  '{"federation-upstream-set": "all"}' --apply-to exchanges

    Shovel vs Federation ΉΕιΊΔζ Ά╕░ΉνΑ:

    • ΈΜρΉΙεΊηΙ ΊΧεΉς╜ΉΩΡΉΕε ΈΜνΈξ╕ Ής╜Ήε╝Έκε ΈσΦΉΜεΉπΑΈξ╝ ΉαχΆ╕░ΈΛΦ Ά▒░Έσ┤ Shovel
    • ΉΩυΈθυ ΈΞ░Ήζ┤ΊΕ░ΉΕ╝ΊΕ░Ήζα exchangeΈξ╝ ΈΖ╝ΈουΉιΒΉε╝Έκε ΊΧσΉ╣αΆ│ι ΉΜ╢Ήε╝Έσ┤ Federation
    • ShovelΉζΑ ΊΧφΉΔΒ ΈσΦΉΜεΉπΑΈξ╝ ΉιΕΈΜυΊΧαΆ│ι, FederationΉζΑ downstreamΉΩΡ ConsumerΆ░Α ΉηΙΉζΕ ΈΧΝΈπΝ ΉιΕΈΜυΊΧεΈΜν

    Management Plugin ΈςρΈΜΙΊΕ░ΈπΒ¶

    rabbitmq-plugins enable rabbitmq_management

    Ά╕░Έ│╕ ΊΠυΊΛ╕ 15672ΉΩΡΉΕε Ήδ╣ UIΈξ╝ ΉιεΆ│╡ΊΧεΈΜν. Ήγ┤ΉαΒΉΩΡΉΕε Έ┤ΡΉΧ╝ ΊΧι ΊΧ╡ΉΜυ ΉπΑΊΣε:

    Ήμ╝ΉγΦ ΈσΦΊΛ╕Έοφ¶

    ΊΒΡ ΈιΙΈ▓ρ:

    • messages_ready: ConsumerΉΩΡΆ▓Ν ΉιΕΈΜυ Ά░ΑΈΛξΊΧε ΈσΦΉΜεΉπΑ ΉΙα. Ά│ΕΉΗΞ ΉοζΆ░ΑΊΧαΈσ┤ ConsumerΆ░Α Έ╢ΑΉκ▒ΊΧαΈΜνΈΛΦ Έε╗.
    • messages_unacknowledged: ConsumerΉΩΡΆ▓Ν ΉιΕΈΜυΈΡΡΉπΑΈπΝ ΉΧΕΉπΒ ack ΉΧΙ ΈΡε ΈσΦΉΜεΉπΑ ΉΙα. Ήζ┤ Ά░ΤΉζ┤ prefetchΉβΑ Ά░βΉε╝Έσ┤ ConsumerΆ░Α Ή▓αΈου ΊΧεΆ│ΕΉΩΡ ΈΠΕΈΜυΊΧε Ά▓Δ.
    • message_bytes: ΊΒΡΆ░Α ΉιΡΉειΊΧαΈΛΦ ΈσΦΈςρΈου/ΈΦΦΉΛνΊΒυ ΊΒυΆ╕░.

    ΈΖ╕ΈΥε ΈιΙΈ▓ρ:

    • mem_used / mem_limit: ΈσΦΈςρΈου ΉΓυΉγσΈθΚΆ│╝ ΉηΕΆ│ΕΆ░Τ.
    • disk_free / disk_free_limit: ΈΦΦΉΛνΊΒυ ΉΩυΉει Ά│╡Ά░Ε.
    • fd_used / fd_total: ΊΝΝΉζ╝ ΈΦΦΉΛνΊΒυΈο╜ΊΕ░ ΉΓυΉγσΈθΚ. Ά│ιΆ░ΙΈΡαΈσ┤ ΉΔΙ ΉΩ░Ά▓░ΉζΕ Ές╗ Έ░δΈΛΦΈΜν.
    • sockets_used / sockets_total: ΉΗΝΉ╝Υ ΉΓυΉγσΈθΚ.

    Connection/Channel ΈιΙΈ▓ρ:

    • state: running, blocking, blocked, flow ΈΥ▒. blockedΈΓα flowΆ░Α Έ│┤Ήζ┤Έσ┤ Έυ╕Ήιε.
    • channels: ConnectionΈΜ╣ Channel ΉΙα. Έ╣ΕΉιΧΉΔΒΉιΒΉε╝Έκε ΈπΟΉε╝Έσ┤ Channel leakΉζΕ ΉζαΉΜυ.

    RetryTemplate ΉΕνΉιΧ¶

    Consumer Ή▓αΈου ΉΜνΊΝρ ΉΜε ΉηυΉΜεΈΠΕ ΈκεΉπΒΉζΕ Spring ΈιΙΈ▓ρΉΩΡΉΕε Ή▓αΈουΊΧεΈΜν. RabbitMQΉζα requeueΉβΑ ΈΜνΈξ┤Ά▓Ν ΈσΦΉΜεΉπΑΈξ╝ ΊΒΡΉΩΡ ΈΠΝΈινΈ│┤ΈΓ┤ΉπΑ ΉΧΛΆ│ι Consumer ΊΦΕΈκεΉΕ╕ΉΛν ΈΓ┤ΉΩΡΉΕε ΉηυΉΜεΈΠΕΊΧεΈΜν.

    @Configuration
