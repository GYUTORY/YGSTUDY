    • access_log: нҒҙлқјмқҙм–ёнҠё мҡ”мІӯмқҙ мІҳлҰ¬лҗ  л•Ңл§ҲлӢӨ н•ң мӨ„м”© кё°лЎқ

    кё°ліё м„Өм •мқҖ мқҙл ҮлӢӨ.

    error_log л ҲлІЁ¶

    error_log /var/log/nginx/error.log warn;

    л ҲлІЁмқҖ debug, info, notice, warn, error, crit, alert, emerg мҲңм„ңлӢӨ. warnмңјлЎң м„Өм •н•ҳл©ҙ warn мқҙмғҒл§Ң кё°лЎқн•ңлӢӨ. мҡҙмҳҒ нҷҳкІҪм—җм„ңлҠ” warnмқҙлӮҳ errorлҘј м“°кі , л¬ём ң м¶”м Ғн•  л•Ңл§Ң infoлЎң лӮ®м¶ҳлӢӨ. debugлҠ” --with-debug мҳөм…ҳмңјлЎң м»ҙнҢҢмқјн•ң Nginxм—җм„ңл§Ң лҸҷмһ‘н•ңлӢӨ.

    м»ӨмҠӨн…Җ лЎңк·ё нҸ¬л§·¶

    кё°ліё combined нҸ¬л§·мқҖ мқҙл ҮкІҢ мғқкІјлӢӨ.

    к°Ғ ліҖмҲҳ мқҳлҜё:

    ліҖмҲҳ м„ӨлӘ…
    $request_time нҒҙлқјмқҙм–ёнҠё мҡ”мІӯ мҲҳмӢ л¶Җн„° мқ‘лӢө мҷ„лЈҢк№Ңм§Җ кұёлҰ° м „мІҙ мӢңк°„(мҙҲ)
    $upstream_connect_time upstream м„ңлІ„м—җ TCP м—°кІ° л§әлҠ” лҚ° кұёлҰ° мӢңк°„
    $upstream_header_time upstreamм—җм„ң мқ‘лӢө н—ӨлҚ” л°ӣкё°к№Ңм§Җ кұёлҰ° мӢңк°„
    $upstream_response_time upstream мқ‘лӢө м „мІҙлҘј л°ӣлҠ” лҚ° кұёлҰ° мӢңк°„
    $request_id Nginxк°Җ мҡ”мІӯл§ҲлӢӨ мһҗлҸҷ мғқм„ұн•ҳлҠ” кі мң  ID (1.11.0+)

    $request_timeкіј $upstream_response_timeмқҳ м°Ёмқҙк°Җ нҒ¬л©ҙ NginxмҷҖ нҒҙлқјмқҙм–ёнҠё мӮ¬мқҙ л„ӨнҠёмӣҢнҒ¬к°Җ лҠҗлҰ° кІғмқҙлӢӨ. $upstream_connect_timeмқҙ нҒ¬л©ҙ upstream м„ңлІ„к°Җ м»Өл„Ҙм…ҳ мҲҳмҡ©мқ„ лӘ» н•ҳкі  мһҲлҠ” кІғмқҙлӢӨ.

    JSON кө¬мЎ°нҷ” лЎңк№…¶

    ELK, Loki, Datadog к°ҷмқҖ лЎңк·ё мҲҳм§‘ мӢңмҠӨн…ңмқ„ м“ҙлӢӨл©ҙ JSON нҸ¬л§·мқҙ нҢҢмӢұн•ҳкё° нҺён•ҳлӢӨ.

    мЈјмқҳн•  м җ: $upstream_response_timeмқҖ upstreamмқҙ м—ҶлҠ” мҡ”мІӯ(м •м Ғ нҢҢмқј м§Ғм ‘ м„ңл№ҷ)м—җм„ң -к°Җ лҗңлӢӨ. мҲ«мһҗ н•„л“ңлЎң л„Јмңјл©ҙ JSON нҢҢмӢұмқҙ мӢӨнҢЁн•ҳлҜҖлЎң, л¬ёмһҗм—ҙ("$upstream_response_time")лЎң к°җмӢём•ј н•ңлӢӨ. $statusлӮҳ $body_bytes_sentлҠ” н•ӯмғҒ мҲ«мһҗк°Җ ліҙмһҘлҗҳлҜҖлЎң л”°мҳҙн‘ң м—Ҷмқҙ мҚЁлҸ„ лҗңлӢӨ.

    мЎ°кұҙл¶Җ лЎңк№… (map)¶

    н—¬мҠӨмІҙнҒ¬ мҡ”мІӯмқҙ мҙҲлӢ№ мҲҳмӢӯ лІҲ л“Өм–ҙмҳӨл©ҙ лЎңк·ёк°Җ л№ лҘҙкІҢ мҢ“мқёлӢӨ. mapмқ„ мҚЁм„ң нҠ№м • мҡ”мІӯмқ„ лЎңк·ём—җм„ң м ңмҷён•ңлӢӨ.

    ~^/ping     0;
    default     1;
}

    нҠ№м • мғҒнғң мҪ”л“ңл§Ң лі„лҸ„ лЎңк·ёлЎң л¶„лҰ¬¶

    лӮҙл¶Җ нҠёлһҳн”Ҫ м ңмҷё¶

    лӘЁлӢҲн„°л§Ғ мӢңмҠӨн…ңмқҙлӮҳ лЎңл“ңл°ёлҹ°м„ңм—җм„ң мҳӨлҠ” мҡ”мІӯмқ„ м ңмҷён•ҳл Өл©ҙ:

    Nginx лЎңк·ё нҢҢмқјмқҖ мһҗлҸҷмңјлЎң лЎңн…Ңмқҙм…ҳлҗҳм§Җ м•ҠлҠ”лӢӨ. л°©м№ҳн•ҳл©ҙ л””мҠӨнҒ¬лҘј лӢӨ мұ„мҡҙлӢӨ. лҢҖл¶Җл¶„мқҳ лҰ¬лҲ…мҠӨ л°°нҸ¬нҢҗм—җм„ң nginx нҢЁнӮӨм§ҖлҘј м„Өм№ҳн•ҳл©ҙ logrotate м„Өм •мқҙ к°ҷмқҙ л“Өм–ҙк°Җм§Җл§Ң, м§Ғм ‘ м»ҙнҢҢмқјн–Ҳкұ°лӮҳ Docker мқҙлҜём§ҖлҘј м“°л©ҙ мҲҳлҸҷмңјлЎң м„Өм •н•ҙм•ј н•ңлӢӨ.

    delaycompressлҠ” л°”лЎң м§Ғм „ лЎңн…Ңмқҙм…ҳ нҢҢмқјмқҖ м••м¶•н•ҳм§Җ м•ҠлҠ”лӢӨлҠ” лң»мқҙлӢӨ. лЎңн…Ңмқҙм…ҳ м§Ғнӣ„м—җ мқҙм „ лЎңк·ёлҘј л°”лЎң нҷ•мқён•ҙм•ј н•  л•Ңк°Җ мһҲм–ҙм„ң л„ЈлҠ”лӢӨ.

    • л“ңлЎӯлҗң м»Өл„Ҙм…ҳ нҢЁл„җ: мқҙ к°’мқҙ 0мқҙ м•„лӢҲл©ҙ worker_connections н•ңлҸ„м—җ кұёлҰ° кІғмқҙлҜҖлЎң, Grafana Alertмқ„ кұём–ҙл‘”лӢӨ. for: 5m мЎ°кұҙмқ„ л„Јм–ҙм„ң мқјмӢңм Ғ мҠӨнҢҢмқҙнҒ¬м—җ м•ҢлҰјмқҙ мҡёлҰ¬м§Җ м•ҠкІҢ н•ңлӢӨ.

    KQL(Kibana Query Language)лЎң мһҗмЈј м“°лҠ” кІҖмғү нҢЁн„ҙ:

    # 5xx м—җлҹ¬л§Ң
status >= 500

# нҠ№м • URIмқҳ лҠҗлҰ° мҡ”мІӯ

# upstream нғҖмһ„м•„мӣғ мқҳмӢ¬ (upstream мқ‘лӢөмқҙ 30мҙҲ мқҙмғҒ)
upstream_response_time > 30

    лЎңк·ё кё°л°ҳ м„ұлҠҘ лі‘лӘ© м§„лӢЁ¶

    лЎңк·ё мҲҳм§‘ мӢңмҠӨн…ңмқҙ к°–м¶°м ё мһҲмңјл©ҙ м„ұлҠҘ л¬ём ңлҘј мІҙкі„м ҒмңјлЎң м¶”м Ғн•  мҲҳ мһҲлӢӨ. мӢӨл¬ҙм—җм„ң мһҗмЈј л§ҲмЈјм№ҳлҠ” нҢЁн„ҙлі„лЎң м •лҰ¬н•ңлӢӨ.

    мӮ¬лЎҖ 1: нҠ№м • APIм—җм„ң к°„н—җм ҒмңјлЎң мқ‘лӢөмқҙ лҠҗл Өм§ҖлҠ” нҳ„мғҒ¶

    мҰқмғҒ: /api/search м—”л“ңнҸ¬мқёнҠёмқҳ p99 мқ‘лӢө мӢңк°„мқҙ нҸүмҶҢ 200msмқёлҚ°, мҳӨнӣ„ 2~4мӢңм—җ 3мҙҲк№Ңм§Җ м№ҳмҶҹлҠ”лӢӨ.

    л¶„м„қ мҲңм„ң:

    1лӢЁкі„ вҖ” мӢңк°„лҢҖлі„ мқ‘лӢө мӢңк°„ л¶„нҸ¬ нҷ•мқё

    Kibanaм—җм„ң лӢӨмқҢ мЎ°кұҙмңјлЎң н•„н„°л§Ғн•ңлӢӨ.

    request_uri: "/api/search*" AND request_time: > 1

    мӢңкі„м—ҙ м°ЁнҠёлЎң мӢңк°„лҢҖлі„ лҠҗлҰ° мҡ”мІӯ мҲҳлҘј нҷ•мқён•ңлӢӨ. нҠ№м • мӢңк°„лҢҖм—җ м§‘мӨ‘лҗҳлҠ”м§Җ, мӮ°л°ңм Ғмқём§ҖлҘј лЁјм Җ кө¬л¶„н•ңлӢӨ.

    2лӢЁкі„ вҖ” upstream мӢңк°„ л¶„м„қ

    лҠҗлҰ° мҡ”мІӯл“Өмқҳ upstream_response_timeкіј request_time м°ЁмқҙлҘј ліёлӢӨ.

    # JSON лЎңк·ём—җм„ң request_timeкіј upstream_response_time м°Ёмқҙк°Җ нҒ° мҡ”мІӯ
cat /var/log/nginx/access.json.log | \
  jq -r 'select(.request_uri | startswith("/api/search")) |
    select(.request_time > 1) |
    "\(.time) rt=\(.request_time) urt=\(.upstream_response_time) \(.request_uri)"'
    • upstream_response_timeмқҙ request_timeкіј л№„мҠ·н•ҳлӢӨ вҶ’ upstream(л°ұм—”л“ң) мһҗмІҙк°Җ лҠҗлҰ° кІғ. DB мҝјлҰ¬лӮҳ мҷёл¶Җ API нҳём¶ңмқ„ нҷ•мқён•ңлӢӨ.
    • upstream_response_timeмқҖ м§§мқҖлҚ° request_timeмқҙ нҒ¬лӢӨ вҶ’ нҒҙлқјмқҙм–ёнҠёлЎң мқ‘лӢөмқ„ ліҙлӮҙлҠ” лҚ° мӢңк°„мқҙ кұёлҰ° кІғ. мқ‘лӢө нҒ¬кё°к°Җ нҒ¬кұ°лӮҳ нҒҙлқјмқҙм–ёнҠё л„ӨнҠёмӣҢнҒ¬к°Җ лҠҗлҰ° кІҪмҡ°лӢӨ.
    • upstream_connect_timeмқҙ нҒ¬лӢӨ вҶ’ upstream м„ңлІ„к°Җ м»Өл„Ҙм…ҳмқ„ мҲҳмҡ© лӘ» н•ҳлҠ” мғҒнғң. upstreamмқҳ backlogлӮҳ worker мҲҳлҘј нҷ•мқён•ңлӢӨ.

    3лӢЁкі„ вҖ” лҸҷмӢң м ‘мҶҚ нҢЁн„ҙ нҷ•мқё

    к°ҷмқҖ мӢңк°„лҢҖмқҳ Active connections мҲҳлҘј stub_status л©”нҠёлҰӯмңјлЎң нҷ•мқён•ңлӢӨ. лҸҷмӢң м ‘мҶҚмқҙ кёүмҰқн•ҳл©ҙм„ң лҠҗл Өм§ҖлҠ” кұҙм§Җ, м ‘мҶҚ мҲҳлҠ” л№„мҠ·н•ңлҚ° нҠ№м • мҡ”мІӯл§Ң лҠҗлҰ° кұҙм§ҖлҘј кө¬л¶„н•ңлӢӨ.

    мӮ¬лЎҖ 2: 502 Bad Gatewayк°Җ мғҲлІҪм—җ н„°м§ҖлҠ” нҳ„мғҒ¶

    мҰқмғҒ: мғҲлІҪ 3~4мӢңм—җ 502 м—җлҹ¬к°Җ мҲҳмӢӯ кұҙ л°ңмғқн•ҳкі , 5л¶„ нӣ„ мһҗлҸҷ ліөкө¬лҗңлӢӨ.
