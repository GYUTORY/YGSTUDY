    OpenTelemetry¶

    к°ңмҡ”¶

    OpenTelemetry(OTel)лҠ” лЎңк·ёВ·л©”нҠёлҰӯВ·нҠёл ҲмқҙмҠӨ м„ё к°Җм§Җ кҙҖм°° лҚ°мқҙн„°лҘј мҲҳм§‘н•ҳкі  лӮҙліҙлӮҙлҠ” лІӨлҚ” мӨ‘лҰҪм Ғ мҳӨн”ҲмҶҢмҠӨ н‘ңмӨҖмқҙлӢӨ. нҠ№м • лӘЁлӢҲн„°л§Ғ лІӨлҚ”м—җ мў…мҶҚлҗҳм§Җ м•Ҡкі , лҚ°мқҙн„°лҘј JaegerВ·TempoВ·PrometheusВ·Datadog л“ұ мӣҗн•ҳлҠ” л°ұм—”л“ңлЎң ліҙлӮј мҲҳ мһҲлӢӨ.

    мҲҳм§‘ лҢҖмғҒ:
  Traces  вҖ” мҡ”мІӯмқҙ м—¬лҹ¬ м„ңл№„мҠӨлҘј кұ°м№ҳлҠ” м „мІҙ кІҪлЎң м¶”м Ғ
  Metrics вҖ” мҲҳм№ҳ кё°л°ҳ мғҒнғң мёЎм • (RPS, мҳӨлҘҳмңЁ, мқ‘лӢө мӢңк°„)
  Logs    вҖ” мқҙлІӨнҠё кё°лЎқ (OTel LogsлҠ” кё°мЎҙ лЎңкұ°мҷҖ м—°лҸҷ)

лӮҙліҙлӮј мҲҳ мһҲлҠ” л°ұм—”л“ң:
  Jaeger / Tempo  вҖ” л¶„мӮ° нҠёл ҲмқҙмӢұ
  Prometheus      вҖ” л©”нҠёлҰӯ
  Loki            вҖ” лЎңк·ё
  Datadog / Dynatrace / New Relic вҖ” мғҒмҡ© APM

    Resource¶

    ResourceлҠ” н…”л Ҳл©”нҠёлҰ¬ лҚ°мқҙн„°лҘј мғқм„ұн•ҳлҠ” м—”нӢ°нӢ°мқҳ м •мІҙм„ұмқҙлӢӨ. "мқҙ лҚ°мқҙн„°к°Җ м–ҙл””м„ң мҷ”лҠ”к°Җ"лҘј лӮҳнғҖлӮҙлҠ” мҶҚм„ұ м§‘н•©мңјлЎң, м„ңл№„мҠӨ мқҙлҰ„, лІ„м „, л°°нҸ¬ нҷҳкІҪ, нҳёмҠӨнҠё м •ліҙ л“ұмқҙ нҸ¬н•ЁлҗңлӢӨ.

    import { Resource } from '@opentelemetry/resources';
import {
  SEMRESATTRS_SERVICE_NAME,
  SEMRESATTRS_SERVICE_VERSION,
} from '@opentelemetry/semantic-conventions';

const resource = new Resource({
  [SEMRESATTRS_SERVICE_NAME]: 'payment-service',

  'service.instance.id': process.env.HOSTNAME ?? 'local',
});

    мӢӨл¬ҙм—җм„ң мһҗмЈј л№ лңЁлҰ¬лҠ” кІғ:

    • service.namespace вҖ” к°ҷмқҖ мқҙлҰ„мқҳ м„ңл№„мҠӨк°Җ м—¬лҹ¬ нҢҖм—җ мһҲмқ„ л•Ң кө¬л¶„мқҙ м•Ҳ лҗңлӢӨ. л„Өмһ„мҠӨнҺҳмқҙмҠӨлҘј л„Јм–ҙм•ј Jaegerм—җм„ң кө¬л¶„лҗңлӢӨ
    • service.instance.id вҖ” мқёмҠӨн„ҙмҠӨлі„ л¬ём ңлҘј м¶”м Ғн•ҳл Өл©ҙ н•„мҲҳлӢӨ. Kubernetesлқјл©ҙ Pod мқҙлҰ„мқ„ л„ЈлҠ”лӢӨ
    • deployment.environment вҖ” stagingкіј production лҚ°мқҙн„°к°Җ к°ҷмқҖ л°ұм—”л“ңм—җ л“Өм–ҙк°Ҳ л•Ң мқҙкұёлЎң н•„н„°л§Ғн•ңлӢӨ

    Semantic Conventions¶

    Semantic ConventionsлҠ” мҶҚм„ұ мқҙлҰ„м—җ лҢҖн•ң кіөнҶө л„Өмқҙл°Қ к·ңм№ҷмқҙлӢӨ. нҢҖл§ҲлӢӨ HTTP мғҒнғң мҪ”л“ңлҘј status_code, http_status, response_code л“ұ лӢӨлҘҙкІҢ м“°л©ҙ лҢҖмӢңліҙл“ңмҷҖ мҝјлҰ¬лҘј м„ңл№„мҠӨлі„лЎң л”°лЎң л§Ңл“Өм–ҙм•ј н•ңлӢӨ. OTelмқҖ мқҙкұё http.response.status_codeлЎң нҶөмқјн•ңлӢӨ.

    мЈјмҡ” л„Өмқҙл°Қ нҢЁн„ҙ:

    http.request.method          вҖ” GET, POST л“ұ
http.response.status_code    вҖ” 200, 500 л“ұ
http.route                   вҖ” /api/users/:id (нҢЁн„ҙ)
url.full                     вҖ” https://api.example.com/users/123

db.system                    вҖ” postgresql, mysql, redis
db.statement                 вҖ” SELECT * FROM users WHERE id = $1
db.operation.name            вҖ” SELECT, INSERT

messaging.system             вҖ” kafka, rabbitmq
messaging.operation.type     вҖ” publish, receive
messaging.destination.name   вҖ” orders-topic

rpc.system                   вҖ” grpc
rpc.method                   вҖ” GetUser
rpc.service                  вҖ” UserService

OTel SDK лІ„м „мқ„ мҳ¬лҰҙ л•Ң мҶҚм„ұ мқҙлҰ„мқҙ лӢ¬лқјм ём„ң кё°мЎҙ лҢҖмӢңліҙл“ңк°Җ к№Ём§Ҳ мҲҳ мһҲмңјлӢҲ, Collectorмқҳ transform processorлЎң л§Ҳмқҙк·ёл Ҳмқҙм…ҳ кё°к°„м—җ м–‘мӘҪ мқҙлҰ„мқ„ лӘЁл‘җ лӮҙліҙлӮҙлҠ” л°©лІ•мқ„ м“°кё°лҸ„ н•ңлӢӨ.

                   к°ҷмқҖ мҡ”мІӯм—җм„ң л§Ңл“Өм–ҙм§„ лӘЁл“  spanмқҙ мқҙ к°’мқ„ кіөмң н•ңлӢӨ

parent_span_id   вҖ” л¶ҖлӘЁ spanмқҳ ID. root spanмқҙл©ҙ л№„м–ҙ мһҲлӢӨ
                   мқҙ к°’мңјлЎң span к°„ нҠёлҰ¬ кө¬мЎ°к°Җ л§Ңл“Өм–ҙм§„лӢӨ

name             вҖ” span мқҙлҰ„. "HTTP GET /users", "DB SELECT users" к°ҷмқҖ нҳ•нғң
                   лӮҳмӨ‘м—җ Jaegerм—җм„ң кІҖмғүн•ҳлҠ” лӢЁмң„мқҙлҜҖлЎң ліҖмҲҳк°’мқ„ л„Јмңјл©ҙ м•Ҳ лҗңлӢӨ

end_time         вҖ” span мў…лЈҢ мӢңк°Ғ. end()лҘј нҳём¶ңн•ҙм•ј кё°лЎқлҗңлӢӨ

                   м—җлҹ¬к°Җ м•„лӢҢ н•ң UNSETмңјлЎң л‘җл©ҙ лҗңлӢӨ. OKлҘј лӘ…мӢңм ҒмңјлЎң м°Қмқ„ н•„мҡ”лҠ” м—ҶлӢӨ

                   мҳҲмҷё л°ңмғқ, мһ¬мӢңлҸ„ мӢңмһ‘ к°ҷмқҖ мқҙлІӨнҠёлҘј нғҖмһ„мҠӨнғ¬н”„мҷҖ н•Ёк»ҳ лӮЁкёҙлӢӨ

links            вҖ” лӢӨлҘё traceлӮҳ spanкіјмқҳ м°ёмЎ° кҙҖкі„
                   л¶ҖлӘЁ-мһҗмӢқмқҙ м•„лӢҢ "кҙҖл Ё мһҲмқҢ" мҲҳмӨҖмқҳ м—°кІ°м—җ мӮ¬мҡ©н•ңлӢӨ

    Events вҖ” Span м•Ҳмқҳ мӢңм җ кё°лЎқ¶

    Spanмқҙ "кө¬к°„"мқ„ лӮҳнғҖлӮёлӢӨл©ҙ, EventлҠ” к·ё кө¬к°„ м•Ҳмқҳ нҠ№м • мӢңм җмқ„ лӮҳнғҖлӮёлӢӨ. мҳҲмҷёк°Җ л°ңмғқн–Ҳкұ°лӮҳ, мһ¬мӢңлҸ„лҘј н–Ҳкұ°лӮҳ, мәҗмӢңлҘј нһҲнҠё/лҜёмҠӨн•ң мҲңк°„мқ„ кё°лЎқн•  л•Ң м“ҙлӢӨ.

    span.addEvent('cache.miss', {
  'cache.key': 'user:123',
});

// мҳҲмҷё кё°лЎқлҸ„ лӮҙл¶Җм ҒмңјлЎң EventлӢӨ
span.recordException(new Error('connection timeout'));
// вҶ’ name: "exception", attributes: { "exception.message": "connection timeout", ... }

    recordException()мқҖ addEvent('exception', ...)мқҳ нҺёмқҳ л©”м„ңл“ңлӢӨ. мҳҲмҷёлҘј кё°лЎқн•ҳл©ҙ Jaegerм—җм„ң н•ҙлӢ№ spanм—җ м—җлҹ¬ л§ҲнҒ¬к°Җ н‘ңмӢңлҗңлӢӨ.

    Links вҖ” мқёкіј кҙҖкі„к°Җ м•„лӢҢ м—°кІ°¶

    л¶ҖлӘЁ-мһҗмӢқ кҙҖкі„лҠ” "Aк°Җ BлҘј нҳём¶ңн–ҲлӢӨ"лҠ” мқёкіј кҙҖкі„лӢӨ. л°ҳл©ҙ LinkлҠ” мқёкіј кҙҖкі„ м—Ҷмқҙ "кҙҖл Ё мһҲмқҢ"мқ„ лӮҳнғҖлӮёлӢӨ.

    Linkк°Җ н•„мҡ”н•ң мғҒнҷ©:

    • л°°м№ҳ мІҳлҰ¬ вҖ” н•ҳлӮҳмқҳ consumer spanмқҙ м—¬лҹ¬ producer spanм—җм„ң мҳЁ л©”мӢңм§ҖлҘј мІҳлҰ¬н•  л•Ң, к°Ғ producer spanмқ„ linkлЎң м—°кІ°н•ңлӢӨ. л¶ҖлӘЁлҠ” н•ҳлӮҳл°–м—җ лӘ» л‘җлӢҲк№Ң linkлҘј м“ҙлӢӨ
    • мһ¬мІҳлҰ¬ вҖ” мӢӨнҢЁн•ң мҡ”мІӯмқ„ мһ¬мІҳлҰ¬н•  л•Ң, мӣҗлһҳ traceлҘј linkлЎң кұём–ҙл‘җл©ҙ "мқҙкұҙ м Җ мҡ”мІӯмқҳ мһ¬мӢңлҸ„"лқјлҠ” л§ҘлқҪмқҙ лӮЁлҠ”лӢӨ
    const link = {
  context: originalSpan.spanContext(),
  attributes: { 'link.reason': 'retry' },
};

tracer.startActiveSpan('order.retry', { links: [link] }, (span) => {
  // мһ¬мІҳлҰ¬ лЎңм§Ғ
  span.end();
});

    Span к°„ кҙҖкі„ вҖ” нҠёлҰ¬мҷҖ DAG¶

    мқјл°ҳм Ғмқё лҸҷкё° нҳём¶ңм—җм„ңлҠ” spanл“Өмқҙ нҠёлҰ¬ кө¬мЎ°лҘј нҳ•м„ұн•ңлӢӨ. н•ҳлӮҳмқҳ л¶ҖлӘЁк°Җ м—¬лҹ¬ мһҗмӢқмқ„ к°Җм§Җкі , мһҗмӢқмқҖ л¶ҖлӘЁлҘј н•ҳлӮҳл§Ң к°Җм§„лӢӨ.

    л№„лҸҷкё° л©”мӢңм§•мқҙлӮҳ л°°м№ҳ мІҳлҰ¬к°Җ м„һмқҙл©ҙ Link л•Ңл¬ём—җ кҙҖкі„к°Җ DAG(Directed Acyclic Graph)лЎң нҷ•мһҘлҗңлӢӨ. л¶ҖлӘЁ-мһҗмӢқ кҙҖкі„лҠ” м—¬м „нһҲ нҠёлҰ¬м§Җл§Ң, LinkлҘј нҸ¬н•Ён•ҳл©ҙ н•ҳлӮҳмқҳ spanмқҙ м—¬лҹ¬ traceмқҳ spanл“Өкіј м—°кІ°лҗ  мҲҳ мһҲлӢӨ.

    Trace 1: [Producer A] в”Җв”Җpublishв”Җв”Җв–¶ [message queue]
Trace 2: [Producer B] в”Җв”Җpublishв”Җв”Җв–¶ [message queue]
                                        в”Ӯ
Trace 3: [Consumer] в—Җв”Җв”Җconsumeв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
          links: [Producer Aмқҳ span, Producer Bмқҳ span]

    Jaegerм—җм„ң Linkк°Җ мһҲлҠ” spanмқ„ нҒҙлҰӯн•ҳл©ҙ м—°кІ°лҗң traceлЎң м җн”„н•  мҲҳ мһҲлӢӨ. лӢӨл§Ң лӘЁл“  нҠёл ҲмқҙмӢұ л°ұм—”л“ңк°Җ LinkлҘј мһҳ мӢңк°Ғнҷ”н•ҳм§ҖлҠ” м•ҠлҠ”лӢӨ. TempoлҠ” м§Җмӣҗн•ҳкі , JaegerлҠ” лІ„м „м—җ л”°лқј лӢӨлҘҙлӢӨ.

    ContextмҷҖ Context Propagation¶

    Contextк°Җ лӯ”к°Җ¶

    ContextлҠ” нҳ„мһ¬ мӢӨн–ү мӨ‘мқё мҪ”л“ңмқҳ л¶Җк°Җ м •ліҙлҘј лӢҙлҠ” л¶ҲліҖ к°қмІҙлӢӨ. к°ҖмһҘ мӨ‘мҡ”н•ң м—ӯн• мқҖ "м§ҖкёҲ нҷңм„ұнҷ”лҗң spanмқҙ лӯ”м§Җ"лҘј л“Өкі  лӢӨлӢҲлҠ” кІғмқҙлӢӨ.

    н•ЁмҲҳ Aк°Җ spanмқ„ мӢңмһ‘н•ҳкі  н•ЁмҲҳ BлҘј нҳём¶ңн•ҳл©ҙ, н•ЁмҲҳ BлҠ” м–ҙл–»кІҢ "лӮҙ л¶ҖлӘЁ spanмқҙ AлӢӨ"лқјлҠ” кұё м•„лҠ”к°Җ? л§ӨлІҲ spanмқ„ нҢҢлқјлҜён„°лЎң л„ҳкёё мҲҳлҠ” м—ҶмңјлӢҲк№Ң, OTelмқҖ ContextлқјлҠ” м•”л¬өм Ғ м ҖмһҘмҶҢм—җ нҳ„мһ¬ spanмқ„ л„Јм–ҙл‘җкі , н•ҳмң„ мҪ”л“ңм—җм„ң context.active()лЎң кәјлӮҙ м“ҙлӢӨ.

    н•ЁмҲҳ нҳём¶ң нқҗлҰ„:

handleRequest()                    context = { activeSpan: spanA }
  в””в”Җв”Җ processOrder()               context = { activeSpan: spanA }  вҶҗ мһҗлҸҷ м „нҢҢ
        в””в”Җв”Җ chargePayment()         context = { activeSpan: spanB }  вҶҗ spanBмқҳ parent = spanA
              в””в”Җв”Җ callStripe()      context = { activeSpan: spanC }  вҶҗ spanCмқҳ parent = spanB

    startActiveSpan()мқ„ нҳём¶ңн•ҳл©ҙ мғҲ spanмқ„ л§Ңл“Өкі , к·ё spanмқ„ лӢҙмқҖ мғҲ ContextлҘј нҳ„мһ¬ мӢӨн–ү лІ”мң„м—җ м„Өм •н•ңлӢӨ. мҪңл°ұмқҙ лҒқлӮҳл©ҙ мқҙм „ ContextлЎң ліөмӣҗлҗңлӢӨ.

    н”„лЎңм„ёмҠӨ лӮҙ м „нҢҢ вҖ” Zone, AsyncLocalStorage¶

    к°ҷмқҖ н”„лЎңм„ёмҠӨ м•Ҳм—җм„ң Contextк°Җ л№„лҸҷкё° мҪ”л“ңлҘј л”°лқјк°Җм•ј н•ңлӢӨ. Node.jsм—җм„ңлҠ” AsyncLocalStorageк°Җ мқҙ м—ӯн• мқ„ н•ңлӢӨ. OTel Node.js SDKлҠ” лӮҙл¶Җм ҒмңјлЎң AsyncLocalStorageлҘј мӮ¬мҡ©н•ҙм„ң, await мІҙмқёмқ„ л”°лқј Contextк°Җ мһҗлҸҷмңјлЎң м „нҢҢлҗңлӢӨ.

    л¬ём ңлҠ” AsyncLocalStorageмқҳ м¶”м Ғмқҙ лҒҠкё°лҠ” кІҪмҡ°лӢӨ.

    // AsyncLocalStorageк°Җ м¶”м Ғн•ҳлҠ” кІҪмҡ° (context м „нҢҢлҗЁ)
await someAsyncFunction();        // OK
new Promise((resolve) => { ... }) // OK
fs.readFile(path, callback);      // OK (Node.js 16+)

// AsyncLocalStorage м¶”м Ғмқҙ лҒҠкё°лҠ” кІҪмҡ° (context мң мӢӨ)
setTimeout(() => { ... });        // мҪңл°ұмқҙ лӢӨлҘё мӢӨн–ү м»Ён…ҚмҠӨнҠё
setInterval(() => { ... });       // л§Ҳм°¬к°Җм§Җ
EventEmitter.on('event', () => { ... });  // мқҙлІӨнҠё н•ёл“Өлҹ¬

    setTimeout м•Ҳм—җм„ң context.active()лҘј нҳём¶ңн•ҳл©ҙ root contextк°Җ л°ҳнҷҳлҗңлӢӨ. л¶ҖлӘЁ span м •ліҙк°Җ м—ҶмңјлӢҲ кі м•„ spanмқҙ л§Ңл“Өм–ҙм§Җкі , Jaegerм—җм„ң нҠёл ҲмқҙмҠӨк°Җ лҒҠкІЁ ліҙмқёлӢӨ. мқҙл•ҢлҠ” context.with()лЎң лӘ…мӢңм ҒмңјлЎң м „лӢ¬н•ҙм•ј н•ңлӢӨ.

    const ctx = context.active();
setTimeout(() => {
  context.with(ctx, () => {
    // м—¬кё°м„ң context.active() === ctx
  });
}, 100);

    Javaм—җм„ңлҠ” ThreadLocalмқҙ к°ҷмқҖ м—ӯн• мқ„ н•ҳлҠ”лҚ°, мҠӨл Ҳл“ң н’Җм—җм„ң мһ‘м—…мқҙ лӢӨлҘё мҠӨл Ҳл“ңлЎң л„ҳм–ҙк°Җл©ҙ Contextк°Җ мң мӢӨлҗңлӢӨ. к·ёлһҳм„ң ExecutorServiceлҘј Context.taskWrapping(executor)мңјлЎң к°җмӢём•ј н•ңлӢӨ.

    н”„лЎңм„ёмҠӨ к°„ м „нҢҢ вҖ” Propagation¶

    м„ңл№„мҠӨ Aм—җм„ң м„ңл№„мҠӨ BлҘј HTTPлЎң нҳём¶ңн•  л•Ң, ContextлҘј HTTP н—ӨлҚ”м—җ мӢӨм–ҙ ліҙлӮҙм•ј н•ңлӢӨ. мқҙкІҢ Context PropagationмқҙлӢӨ.

    кіјм •мқҖ м„ё лӢЁкі„лӢӨ:

    1. Inject (мЈјмһ…)  вҖ” м„ңл№„мҠӨ Aк°Җ нҳ„мһ¬ Contextм—җм„ң trace_id, span_idлҘј кәјлӮҙ
                    HTTP мҡ”мІӯ н—ӨлҚ”м—җ л„ЈлҠ”лӢӨ
                    propagation.inject(context.active(), headers)

2. Transfer (м „мҶЎ) вҖ” HTTP мҡ”мІӯмқҙ л„ӨнҠёмӣҢнҒ¬лҘј нҶөн•ҙ м„ңл№„мҠӨ BлЎң м „лӢ¬лҗңлӢӨ
                    traceparent: 00-abc123-def456-01 (W3C нҸ¬л§·)

3. Extract (м¶”м¶ң) вҖ” м„ңл№„мҠӨ Bк°Җ мҲҳмӢ н•ң н—ӨлҚ”м—җм„ң trace_id, span_idлҘј кәјлӮҙ
                    мғҲ ContextлҘј л§Ңл“Өкі , мқҙнӣ„ мғқм„ұлҗҳлҠ” spanмқҳ л¶ҖлӘЁлЎң м„Өм •н•ңлӢӨ
                    propagation.extract(context.active(), headers)

    мһҗлҸҷ кі„мёЎ(@opentelemetry/instrumentation-http л“ұ)мқҙ мқҙ inject/extractлҘј м•Ңм•„м„ң мІҳлҰ¬н•ңлӢӨ. HTTP client лқјмқҙлёҢлҹ¬лҰ¬(axios, fetch)лЎң мҡ”мІӯмқ„ ліҙлӮҙл©ҙ мһҗлҸҷмңјлЎң traceparent н—ӨлҚ”к°Җ м¶”к°Җлҗҳкі , Express/Fastify м„ңлІ„м—җм„ң мҡ”мІӯмқ„ л°ӣмңјл©ҙ мһҗлҸҷмңјлЎң н—ӨлҚ”лҘј мқҪм–ҙ л¶ҖлӘЁ spanмқ„ м„Өм •н•ңлӢӨ.

    мҲҳлҸҷмңјлЎң н•ҙм•ј н•ҳлҠ” кІҪмҡ°лҠ” мһҗлҸҷ кі„мёЎмқҙ м§Җмӣҗн•ҳм§Җ м•ҠлҠ” м „мҶЎ мҲҳлӢЁмқ„ м“ё л•ҢлӢӨ. мҳҲлҘј л“Өм–ҙ м»ӨмҠӨн…Җ л©”мӢңм§Җ нҒҗ, gRPC metadata, WebSocket л“ұ.

    Propagator мў…лҘҳмҷҖ м„ нғқ¶

    Propagationм—җм„ң "м–ҙл–Ө нҸ¬л§·мңјлЎң м§Ғл ¬нҷ”н•ҳлҠҗлғҗ"лҘј кІ°м •н•ҳлҠ” кІҢ PropagatorлӢӨ. м•„лһҳ Propagator м„№м…ҳм—җм„ң W3C TraceContext, B3 л“ұ кө¬мІҙм Ғмқё нҸ¬л§·мқ„ лӢӨлЈ¬лӢӨ.

    Sampler мў…лҘҳмҷҖ лҸҷмһ‘ мӣҗлҰ¬¶

    нҳ„мһ¬ л¬ём„ңмқҳ мғҳн”Ңл§Ғ м„№м…ҳм—җм„ң head/tail кө¬л¶„л§Ң к°„лһөнһҲ лӢӨлӨҳлҠ”лҚ°, SDKм—җм„ң м„Өм •н•ҳлҠ” Samplerмқҳ кө¬мІҙм Ғмқё мў…лҘҳмҷҖ мЎ°н•© л°©мӢқмқҙ л№ м ё мһҲлӢӨ.

    Head-based Sampler (SDKм—җм„ң м„Өм •)¶

    Head-based мғҳн”Ңл§ҒмқҖ spanмқҙ л§Ңл“Өм–ҙм§ҖлҠ” мӢңм җм—җ мҲҳм§‘ м—¬л¶ҖлҘј кІ°м •н•ңлӢӨ. SDKмқҳ Sampler мқён„°нҺҳмқҙмҠӨк°Җ мқҙ м—ӯн• мқ„ н•ңлӢӨ.

    AlwaysOnSampler вҖ” лӘЁл“  spanмқ„ мҲҳм§‘н•ңлӢӨ. л””лІ„к№…мқҙлӮҳ нҠёлһҳн”Ҫмқҙ м ҒмқҖ м„ңл№„мҠӨм—җм„ң м“ҙлӢӨ. н”„лЎңлҚ•м…ҳм—җм„ң нҠёлһҳн”Ҫмқҙ л§ҺмқҖ м„ңл№„мҠӨм—җ кұёл©ҙ мҠӨнҶ лҰ¬м§Җ л№„мҡ©мқҙ к°җлӢ№мқҙ м•Ҳ лҗңлӢӨ.

    AlwaysOffSampler вҖ” м•„л¬ҙкІғлҸ„ мҲҳм§‘н•ҳм§Җ м•ҠлҠ”лӢӨ. span мһҗмІҙлҠ” мғқм„ұлҗҳм§Җл§Ң NoOp мғҒнғңлқј мҶҚм„ұмқҙлӮҳ мқҙлІӨнҠёк°Җ кё°лЎқлҗҳм§Җ м•Ҡкі  exportлҸ„ м•Ҳ лҗңлӢӨ. нҠ№м • м„ңл№„мҠӨмқҳ нҠёл ҲмқҙмӢұмқ„ мҷ„м „нһҲ лҒ„кі  мӢ¶мқ„ л•Ң м“ҙлӢӨ.

    TraceIdRatioBasedSampler вҖ” trace IDмқҳ н•ҙмӢңк°’мқ„ кё°мӨҖмңјлЎң л№„мңЁ мғҳн”Ңл§Ғн•ңлӢӨ. ratio: 0.1мқҙл©ҙ 10%мқҳ traceлҘј мҲҳм§‘н•ңлӢӨ. мӨ‘мҡ”н•ң м җмқҖ, к°ҷмқҖ trace IDлҠ” м–ҙл–Ө м„ңл№„мҠӨм—җм„ңл“  к°ҷмқҖ кІ°м •мқҙ лӮҳмҳЁлӢӨлҠ” кІғмқҙлӢӨ. trace ID мһҗмІҙк°Җ лһңлҚӨмқҙкі , к·ё к°’мқҳ н•ҳмң„ л№„нҠёлҘј л№„мңЁкіј л№„көҗн•ҳкё° л•Ңл¬ёмқҙлӢӨ. к·ёлһҳм„ң м„ңл№„мҠӨ Aм—җм„ң мғҳн”Ңл§Ғлҗң traceлҠ” м„ңл№„мҠӨ Bм—җм„ңлҸ„ мғҳн”Ңл§ҒлҗңлӢӨ(к°ҷмқҖ ratioлҘј м“ҙлӢӨл©ҙ).

    import { TraceIdRatioBasedSampler } from '@opentelemetry/sdk-trace-base';

// 10% мғҳн”Ңл§Ғ вҖ” trace ID н•ҙмӢң кё°л°ҳмқҙлқј м„ңл№„мҠӨ к°„ мқјкҙҖм„ұ ліҙмһҘ
const sampler = new TraceIdRatioBasedSampler(0.1);

    ParentBasedSampler вҖ” л¶ҖлӘЁ spanмқҳ мғҳн”Ңл§Ғ кІ°м •мқ„ л”°лҘёлӢӨ. л¶ҖлӘЁк°Җ мғҳн”Ңл§Ғлҗҗмңјл©ҙ мһҗмӢқлҸ„ мғҳн”Ңл§Ғн•ҳкі , м•Ҳ лҗҗмңјл©ҙ мһҗмӢқлҸ„ м•Ҳ н•ңлӢӨ. л¶ҖлӘЁк°Җ м—ҶлҠ” root spanм—җ лҢҖн•ҙм„ңл§Ң лі„лҸ„ SamplerлҘј м Ғмҡ©н•ңлӢӨ.

    мқҙкІҢ мҷң н•„мҡ”н•ңк°Җ: м„ңл№„мҠӨ Aк°Җ 10% мғҳн”Ңл§ҒмңјлЎң traceлҘј мӢңмһ‘н–ҲлҠ”лҚ°, м„ңл№„мҠӨ Bк°Җ лі„лҸ„лЎң 20% мғҳн”Ңл§Ғмқ„ н•ҳл©ҙ Aм—җм„ңлҠ” мҲҳм§‘лҗҳлҠ” traceк°Җ Bм—җм„ңлҠ” лІ„л Өм§ҖлҠ” кІҪмҡ°к°Җ мғқкёҙлӢӨ. traceк°Җ мӨ‘к°„м—җ лҒҠкё°лҠ” кІғмқҙлӢӨ. ParentBasedSamplerлҠ” upstreamмқҳ кІ°м •мқ„ мЎҙмӨ‘н•ҙм„ң мқҙ л¬ём ңлҘј л°©м§Җн•ңлӢӨ.

    import { ParentBasedSampler, TraceIdRatioBasedSampler, AlwaysOnSampler } from '@opentelemetry/sdk-trace-base';

const sampler = new ParentBasedSampler({
  root: new TraceIdRatioBasedSampler(0.1),         // root spanмқҖ 10% мғҳн”Ңл§Ғ
  remoteParentSampled: new AlwaysOnSampler(),       // upstreamмқҙ мҲҳм§‘ кІ°м • вҶ’ л”°лҘёлӢӨ
  remoteParentNotSampled: new AlwaysOffSampler(),   // upstreamмқҙ лҜёмҲҳм§‘ кІ°м • вҶ’ л”°лҘёлӢӨ
  localParentSampled: new AlwaysOnSampler(),        // к°ҷмқҖ н”„лЎңм„ёмҠӨ лӮҙ л¶ҖлӘЁк°Җ мҲҳм§‘ вҶ’ л”°лҘёлӢӨ
  localParentNotSampled: new AlwaysOffSampler(),    // к°ҷмқҖ н”„лЎңм„ёмҠӨ лӮҙ л¶ҖлӘЁк°Җ лҜёмҲҳм§‘ вҶ’ л”°лҘёлӢӨ
});

    NodeSDKмқҳ кё°ліё SamplerлҠ” ParentBasedSampler(root: AlwaysOnSampler)лӢӨ. мҰү, root spanмқҖ м „л¶Җ мҲҳм§‘н•ҳкі , л¶ҖлӘЁк°Җ мһҲмңјл©ҙ л¶ҖлӘЁ кІ°м •мқ„ л”°лҘёлӢӨ. н”„лЎңлҚ•м…ҳм—җм„ң rootмқҳ AlwaysOnмқ„ TraceIdRatioлЎң л°”кҫёлҠ” кІҢ мқјл°ҳм ҒмқҙлӢӨ.

    мЎ°н•© л°©мӢқ¶

    мӢӨл¬ҙм—җм„ң м“°лҠ” нҢЁн„ҙ:

    нҢЁн„ҙ 1: ParentBased + TraceIdRatio (к°ҖмһҘ нқ”н•Ё)
  вҶ’ root spanмқҖ 10% мғҳн”Ңл§Ғ, лӮҳлЁём§ҖлҠ” л¶ҖлӘЁлҘј л”°лҰ„
  вҶ’ м„ңл№„мҠӨ к°„ traceк°Җ лҒҠкё°м§Җ м•ҠмқҢ

нҢЁн„ҙ 2: м„ңл№„мҠӨлі„ лӢӨлҘё л№„мңЁ
  вҶ’ кІ°м ң м„ңл№„мҠӨ: 100% (мһҘм•  м¶”м Ғ мӨ‘мҡ”)
  вҶ’ мғҒн’Ҳ лӘ©лЎқ м„ңл№„мҠӨ: 1% (нҠёлһҳн”Ҫ л§Һкі  лҢҖл¶Җл¶„ м •мғҒ)
  вҶ’ ParentBasedлҠ” к·ёлҢҖлЎң м Ғмҡ©н•ҙм„ң upstream кІ°м •мқ„ мЎҙмӨ‘

нҢЁн„ҙ 3: Head + Tail мЎ°н•©
  вҶ’ SDK(Head): ParentBased + TraceIdRatio(0.2)лЎң 20% мҲҳм§‘
  вҶ’ Collector(Tail): к·ё 20% мӨ‘м—җм„ң м—җлҹ¬/лҠҗлҰ° кІғл§Ң мөңмў… м ҖмһҘ
  вҶ’ мҲҳм§‘ л№„мҡ©кіј мһҘм•  м¶”м Ғ мӮ¬мқҙмқҳ к· нҳ•м җ

    SamplerлҠ” SDK(head-based)мҷҖ Collector(tail-based)м—җм„ң к°Ғк°Ғ лӢӨлҘё л Ҳмқҙм–ҙлЎң лҸҷмһ‘н•ңлӢӨ. SDK SamplerлҠ” span мғқм„ұ мӢңм җм—җ кІ°м •н•ҳкі , Collectorмқҳ tail_sampling processorлҠ” trace м „мІҙк°Җ лҸ„м°©н•ң нӣ„м—җ кІ°м •н•ңлӢӨ. л‘ҳмқ„ мЎ°н•©н•ҳл©ҙ SDKм—җм„ң лҢҖлһөм ҒмңјлЎң кұёлҹ¬лӮҙкі , Collectorм—җм„ң м •л°Җн•ҳкІҢ м„ лі„н•ҳлҠ” 2лӢЁ кө¬мЎ°к°Җ лҗңлӢӨ.

    мқҙкІҢ SDKлҘј лӢӨлҘё лӘЁл“ҲліҙлӢӨ лЁјм Җ мҙҲкё°нҷ”н•ҙм•ј н•ҳлҠ” мқҙмң лӢӨ. require('http')к°Җ лЁјм Җ мӢӨн–үлҗҳл©ҙ мӣҗліё лӘЁл“Ҳмқҙ мқҙлҜё мәҗмӢңм—җ мҳ¬лқјк°„ мғҒнғңлқј нҢЁм№ҳн•  нғҖмқҙл°Қмқ„ лҶ“м№ңлӢӨ. Node.jsмқҳ require мәҗмӢң нҠ№м„ұмғҒ, н•ңлІҲ лЎңл“ңлҗң лӘЁл“ҲмқҖ лӢӨмӢң лЎңл“ңн•ҳм§Җ м•ҠлҠ”лӢӨ.

    Java Agent(-javaagent:opentelemetry-javaagent.jar)лҠ” лҚ” м ҖмҲҳмӨҖм—җм„ң лҸҷмһ‘н•ңлӢӨ. JVMмқҳ bytecode instrumentationмқ„ мӮ¬мҡ©н•ҙм„ң нҒҙлһҳмҠӨ лЎңл”© мӢңм җм—җ л°”мқҙнҠёмҪ”л“ңлҘј мҲҳм •н•ңлӢӨ. к·ёлһҳм„ң мҪ”л“ң мҲҳм • м—Ҷмқҙ JVM мҳөм…ҳ н•ҳлӮҳлЎң кі„мёЎмқҙ к°ҖлҠҘн•ҳлӢӨ.

    Auto vs Manual нҠёл Ҳмқҙл“ңмҳӨн”„¶

    Auto Instrumentation:
  мһҘм җ
  - мҪ”л“ң мҲҳм • м—Ҷмқҙ HTTP, DB, л©”мӢңм§Җ нҒҗ л“ұ мқён”„лқј л ҲлІЁ кі„мёЎмқҙ лҗңлӢӨ
  - лқјмқҙлёҢлҹ¬лҰ¬ м—…лҚ°мқҙнҠё мӢң кі„мёЎлҸ„ мһҗлҸҷмңјлЎң л§һм¶°м§„лӢӨ
  - м„Өм • лӘҮ мӨ„лЎң м„ңл№„мҠӨ к°„ нҳём¶ң кҙҖкі„к°Җ Jaegerм—җ ліҙмқёлӢӨ

  н•ңкі„
  - л№„мҰҲлӢҲмҠӨ лЎңм§Ғмқҳ мқҳлҜёлҘј лӘЁлҘёлӢӨ
    "кІ°м ң мҠ№мқё лӢЁкі„"мҷҖ "нҸ¬мқёнҠё м ҒлҰҪ лӢЁкі„"лҘј кө¬л¶„ лӘ» н•ңлӢӨ
    DB мҝјлҰ¬к°Җ лҠҗлҰ° кұҙ ліҙмқҙм§Җл§Ң, мҷң к·ё мҝјлҰ¬к°Җ мӢӨн–үлҗҗлҠ”м§ҖлҠ” лӘЁлҘёлӢӨ
  - мҶҚм„ұмқ„ л§ҲмқҢлҢҖлЎң лӘ» л¶ҷмқёлӢӨ
    order_id, user_tier к°ҷмқҖ л№„мҰҲлӢҲмҠӨ мҶҚм„ұмқҙ м—Ҷмңјл©ҙ
    "лҠҗлҰ° мҡ”мІӯмқҙ лҲ„кө¬ кұҙм§Җ" нҢҢм•…н•  мҲҳк°Җ м—ҶлӢӨ
  - monkey-patchingмқҙлқј лқјмқҙлёҢлҹ¬лҰ¬ лӮҙл¶Җ кө¬нҳ„мқҙ л°”лҖҢл©ҙ к№Ём§Ҳ мҲҳ мһҲлӢӨ
    мӢӨм ңлЎң OTel SDK лІ„м „кіј кі„мёЎ лҢҖмғҒ лқјмқҙлёҢлҹ¬лҰ¬ лІ„м „ мӮ¬мқҙм—җ
    нҳёнҷҳм„ұ л¬ём ңк°Җ мғқкё°лҠ” кІҪмҡ°к°Җ мһҲлӢӨ

Manual Instrumentation:
  мһҘм җ
  - л№„мҰҲлӢҲмҠӨ лЎңм§Ғ лӢЁмң„лЎң spanмқ„ л§Ңл“Ө мҲҳ мһҲлӢӨ
    "мһҘл°”кө¬лӢҲ кі„мӮ°", "мҝ нҸ° м Ғмҡ©", "мһ¬кі  нҷ•мқё" к°Ғк°Ғмқ„ spanмңјлЎң л¶„лҰ¬
  - мӣҗн•ҳлҠ” мҶҚм„ұмқ„ мһҗмң лЎӯкІҢ м¶”к°Җн•  мҲҳ мһҲлӢӨ
  - мҪ”л“ңм—җм„ң м§Ғм ‘ м ңм–ҙн•ҳлӢҲк№Ң лҸҷмһ‘мқҙ мҳҲмёЎ к°ҖлҠҘн•ҳлӢӨ

  н•ңкі„
  - мҪ”л“ңм—җ OTel мқҳмЎҙм„ұмқҙ л“Өм–ҙк°„лӢӨ
  - span мӢңмһ‘/мў…лЈҢлҘј л№ лңЁлҰ¬л©ҙ нҠёл ҲмқҙмҠӨк°Җ к№Ём§„лӢӨ
  - мҪ”л“ң ліҖкІҪмқҙ н•„мҡ”н•ҳлӢӨ

    мӢӨл¬ҙм—җм„ңлҠ” л‘ҳ лӢӨ м“ҙлӢӨ¶

    AutoлЎң мқён”„лқј л ҲлІЁ(HTTP, DB, нҒҗ)мқ„ к№”кі , л№„мҰҲлӢҲмҠӨ лЎңм§Ғм—җ ManualлЎң м»ӨмҠӨн…Җ spanмқ„ м¶”к°Җн•ҳлҠ” кІҢ мқјл°ҳм ҒмқҙлӢӨ.

    // Auto: Express мҡ”мІӯ м§„мһ…, HTTP нҒҙлқјмқҙм–ёнҠё нҳём¶ң, DB мҝјлҰ¬к°Җ мһҗлҸҷмңјлЎң мһЎнһҢлӢӨ
// Manual: л№„мҰҲлӢҲмҠӨ лЎңм§Ғ лӢЁмң„лҘј м§Ғм ‘ кі„мёЎ

async function processOrder(orderId: string) {
  // мқҙ spanмқҖ Express мҡ”мІӯ spanмқҳ мһҗмӢқмңјлЎң мһҗлҸҷ м—°кІ°лҗңлӢӨ
  return tracer.startActiveSpan('order.process', async (span) => {
    span.setAttribute('order.id', orderId);

    const items = await getOrderItems(orderId);    // auto: DB мҝјлҰ¬ span мһҗлҸҷ мғқм„ұ
    const total = calculateTotal(items);            // manual л¶Ҳн•„мҡ” вҖ” л№ лҘҙкі  лӢЁмҲңн•ң лЎңм§Ғ

    // кІ°м ңлҠ” мӨ‘мҡ”н•ҳлӢҲк№Ң manual span м¶”к°Җ
    await tracer.startActiveSpan('order.payment', async (paySpan) => {
      paySpan.setAttribute('payment.amount', total);
      await chargePayment(orderId, total);          // auto: HTTP нҒҙлқјмқҙм–ёнҠё span мһҗлҸҷ мғқм„ұ
      paySpan.end();
    });

    span.end();
  });
}

    OTLP (OpenTelemetry Protocol)¶

    OTLPлҠ” OTelм—җм„ң н…”л Ҳл©”нҠёлҰ¬ лҚ°мқҙн„°лҘј м „мҶЎн•ҳлҠ” н‘ңмӨҖ н”„лЎңнҶ мҪңмқҙлӢӨ. SDKм—җм„ң CollectorлЎң, Collectorм—җм„ң л°ұм—”л“ңлЎң лҚ°мқҙн„°лҘј ліҙлӮј л•Ң мӮ¬мҡ©н•ңлӢӨ.

    м§Ғл ¬нҷ” кө¬мЎ°¶

    OTLPлҠ” Protocol Buffers(protobuf) кё°л°ҳмқҙлӢӨ. .proto нҢҢмқјлЎң л©”мӢңм§Җ мҠӨнӮӨл§Ҳк°Җ м •мқҳлҗҳм–ҙ мһҲкі , traceВ·metricВ·log к°Ғк°Ғм—җ лҢҖн•ң л©”мӢңм§Җ нғҖмһ…мқҙ мһҲлӢӨ. protobuf л°”мқҙл„ҲлҰ¬ м§Ғл ¬нҷ” лҚ•л¶„м—җ JSONліҙлӢӨ нҺҳмқҙлЎңл“ңк°Җ мһ‘кі  нҢҢмӢұмқҙ л№ лҘҙлӢӨ.

    HTTP м „мҶЎм—җм„ңлҠ” protobuf л°”мқҙл„ҲлҰ¬(Content-Type: application/x-protobuf)к°Җ кё°ліёмқҙкі , JSON(application/json)лҸ„ м§Җмӣҗн•ңлӢӨ. JSONмқҖ л””лІ„к№… мҡ©лҸ„лЎң м“°кі , н”„лЎңлҚ•м…ҳм—җм„ңлҠ” protobufлҘј м“ҙлӢӨ.

    м „мҶЎ л°©мӢқ: gRPC vs HTTP¶

    gRPC (нҸ¬нҠё 4317) HTTP (нҸ¬нҠё 4318)
    м§Ғл ¬нҷ” protobuf л°”мқҙл„ҲлҰ¬ protobuf л°”мқҙл„ҲлҰ¬ лҳҗлҠ” JSON
    м»Өл„Ҙм…ҳ HTTP/2 л©ҖнӢ°н”Ңл үмӢұ HTTP/1.1 лҳҗлҠ” HTTP/2
    мҠӨнҠёлҰ¬л°Қ м§Җмӣҗ лҜём§Җмӣҗ
    н”„лЎқмӢң нҶөкіј HTTP/2 м§Җмӣҗ н•„мҡ” лҢҖл¶Җл¶„мқҳ н”„лЎқмӢңм—җм„ң лҸҷмһ‘
    л””лІ„к№… л°”мқҙл„ҲлҰ¬лқј curl л¶Ҳк°Җ JSON лӘЁл“ңлЎң curl к°ҖлҠҘ

    к°ҷмқҖ Kubernetes нҒҙлҹ¬мҠӨн„° м•Ҳмқҙлқјл©ҙ gRPCк°Җ лӮ«лӢӨ. ALBлӮҳ Nginx к°ҷмқҖ L7 н”„лЎқмӢңлҘј кұ°міҗм•ј н•ҳлҠ” нҷҳкІҪмқҙл©ҙ HTTPк°Җ м„Өм •мқҙ лӢЁмҲңн•ҳлӢӨ.

SERVER   вҖ” мҷёл¶Җм—җм„ң л“Өм–ҙмҳЁ мҡ”мІӯмқ„ мІҳлҰ¬н•ҳлҠ” мӘҪ. Express н•ёл“Өлҹ¬, gRPC м„ңлІ„ л©”м„ңл“ң
PRODUCER вҖ” л©”мӢңм§ҖлҘј нҒҗм—җ л„ЈлҠ” мӘҪ. Kafka produce, RabbitMQ publish
CONSUMER вҖ” нҒҗм—җм„ң л©”мӢңм§ҖлҘј кәјлӮҙ мІҳлҰ¬н•ҳлҠ” мӘҪ. Kafka consume, RabbitMQ subscribe
INTERNAL вҖ” м„ңл№„мҠӨ лӮҙл¶Җ лЎңм§Ғ. мҷёл¶Җ нҳём¶ңмқҙ м•„лӢҢ л№„мҰҲлӢҲмҠӨ лЎңм§Ғ span (кё°ліёк°’)

    мһҗлҸҷ кі„мёЎ лқјмқҙлёҢлҹ¬лҰ¬лҠ” мқҙкұё м•Ңм•„м„ң м„Өм •н•ңлӢӨ. Express instrumentationмқҖ л“Өм–ҙмҳЁ мҡ”мІӯм—җ SERVERлҘј, HTTP client instrumentationмқҖ лӮҳк°ҖлҠ” мҡ”мІӯм—җ CLIENTлҘј л¶ҷмқёлӢӨ. м§Ғм ‘ spanмқ„ л§Ңл“Ө л•Ңл§Ң мӢ кІҪ м“°л©ҙ лҗңлӢӨ.

    import { SpanKind } from '@opentelemetry/api';

// лӢӨлҘё м„ңл№„мҠӨлҘј нҳём¶ңн•ҳлҠ” span вҖ” CLIENT
tracer.startActiveSpan('user-service.getUser', { kind: SpanKind.CLIENT }, async (span) => {
  const user = await fetch('http://user-service/api/users/123');
  span.end();
});

// Kafkaм—җ л©”мӢңм§ҖлҘј ліҙлӮҙлҠ” span вҖ” PRODUCER
tracer.startActiveSpan('orders.publish', { kind: SpanKind.PRODUCER }, async (span) => {
  await kafka.producer.send({ topic: 'orders', messages: [{ value: orderJson }] });
  span.end();
});

// м„ңл№„мҠӨ лӮҙл¶Җ кі„мӮ° вҖ” INTERNAL (кё°ліёк°’мқҙлқј мғқлһө к°ҖлҠҘ)
tracer.startActiveSpan('payment.calculateFee', async (span) => {
  const fee = calculateFee(amount);
  span.end();
  return fee;
});

    CLIENTмҷҖ SERVER spanмқҖ м§қмқ„ мқҙлЈ¬лӢӨ. м„ңл№„мҠӨ Aмқҳ CLIENT spanкіј м„ңл№„мҠӨ Bмқҳ SERVER spanмқҙ к°ҷмқҖ trace м•Ҳм—җм„ң л¶ҖлӘЁ-мһҗмӢқ кҙҖкі„лЎң м—°кІ°лҗҳл©ҙ, Jaegerм—җм„ң м„ңл№„мҠӨ к°„ нҳём¶ң кҙҖкі„лҘј к·ёлҰҙ мҲҳ мһҲлӢӨ. PRODUCERмҷҖ CONSUMERлҸ„ л§Ҳм°¬к°Җм§ҖлЎң м§қмқ„ мқҙлЈ¬лӢӨ.

    kindлҘј мһҳлӘ» м§Җм •н•ҳл©ҙ Jaegerмқҳ м„ңл№„мҠӨ мқҳмЎҙм„ұ к·ёлһҳн”„(DAG)к°Җ м—үлҡұн•ҳкІҢ лӮҳмҳЁлӢӨ. лӮҙл¶Җ лЎңм§ҒмқёлҚ° CLIENTлЎң м§Җм •н•ҳл©ҙ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ” мҷёл¶Җ м„ңл№„мҠӨ нҳём¶ңмІҳлҹј ліҙмқёлӢӨ.

    м»ӨмҠӨн…Җ Span вҖ” л№„мҰҲлӢҲмҠӨ лЎңм§Ғ м¶”м Ғ¶

    import { trace, SpanStatusCode, context, propagation } from '@opentelemetry/api';

const tracer = trace.getTracer('payment-service', '1.0.0');

async function processPayment(orderId: string, amount: number): Promise<PaymentResult> {
  return tracer.startActiveSpan('payment.process', async (span) => {
    // Span мҶҚм„ұ (Jaegerм—җм„ң кІҖмғү к°ҖлҠҘ)
    span.setAttributes({
      'order.id': orderId,
      'payment.amount': amount,
      'payment.currency': 'KRW',
    });

    try {
      // мӨ‘мІ© Span вҖ” н•ҳмң„ мһ‘м—… м¶”м Ғ
      const result = await tracer.startActiveSpan('payment.gateway.charge', async (childSpan) => {
        childSpan.setAttribute('gateway', 'stripe');
        const r = await stripeClient.charge(orderId, amount);
        childSpan.setStatus({ code: SpanStatusCode.OK });
        childSpan.end();
        return r;
      });

      span.setStatus({ code: SpanStatusCode.OK });
      span.setAttribute('payment.transaction_id', result.transactionId);
      return result;

    } catch (err) {
      span.setStatus({
        code: SpanStatusCode.ERROR,
        message: (err as Error).message,
      });
      span.recordException(err as Error);
      throw err;
    } finally {
      span.end();
    }
  });
}

    OTel Logs лӘЁлҚё¶

    OTelмқҳ LogsлҠ” Traces, MetricsмҷҖ м ‘к·ј л°©мӢқмқҙ лӢӨлҘҙлӢӨ. TracesмҷҖ MetricsлҠ” OTel APIлЎң м§Ғм ‘ мғқм„ұн•ҳм§Җл§Ң, LogsлҠ” кё°мЎҙ лЎңкұ°(winston, pino, log4j л“ұ)лҘј к·ёлҢҖлЎң м“°л©ҙм„ң OTel нҢҢмқҙн”„лқјмқём—җ м—°кІ°н•ҳлҠ” л°©мӢқмқҙлӢӨ. мқҙкұё Log Bridge APIлқјкі  н•ңлӢӨ.

    мҷң Log Bridgeмқёк°Җ¶

    LogRecord кө¬мЎ°¶

    OTelмқҳ LogRecordлҠ” лӢӨмқҢ н•„л“ңлЎң кө¬м„ұлҗңлӢӨ.

    Timestamp           вҖ” лЎңк·ё л°ңмғқ мӢңк°Ғ (nanosecond м •л°ҖлҸ„)

      span.setAttribute(`baggage.${key}`, entry.value);

  }
}
